============================

============================

Эта документация предназначена для разработчиков и контрибьюторов ядра Linux,
интересующихся внутренним устройством PREEMPT_RT. В ней объясняются ключевые
концепции и изменения, необходимые по сравнению с конфигурацией без PREEMPT_RT.
