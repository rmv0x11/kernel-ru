Руководство по процессу разработки ядра
=======================================

Цель этого документа — помочь разработчикам (и их руководителям) работать с
сообществом разработки с минимумом разочарований. Это попытка описать, как
устроена работа данного сообщества, в форме, доступной для тех, кто не знаком
близко с разработкой ядра Linux (да и со свободной разработкой ПО в целом).
Хотя здесь есть некоторый технический материал, это во многом обсуждение,
ориентированное на процесс, и для его понимания не требуется глубокого знания
программирования ядра.
