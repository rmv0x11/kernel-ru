:Author: Matt Porter

Введение
========

RapidIO — это высокоскоростная коммутируемая фабрика межсоединений (switched
fabric interconnect) с возможностями, ориентированными на рынок встраиваемых
систем. RapidIO обеспечивает поддержку как memory-mapped I/O, так и транзакций
на основе сообщений поверх сети с коммутируемой фабрикой. RapidIO имеет
стандартизированный механизм обнаружения, во многом схожий со стандартом шины
PCI, который позволяет легко обнаруживать устройства в сети.

Эта документация предназначена для разработчиков, которые намереваются добавить
поддержку RapidIO для новых архитектур, написать новые драйверы или понять
внутреннее устройство подсистемы.

Известные ошибки и ограничения
==============================

Ошибки
------

Нет. ;)

Ограничения
-----------

1. Доступ к областям памяти RapidIO и управление ими не поддерживаются

2. Перечисление с несколькими хостами (multiple host enumeration) не поддерживается

Интерфейс драйверов RapidIO
===========================

Драйверам предоставляется набор вызовов для взаимодействия с подсистемой с целью
сбора сведений об устройствах, запроса/отображения ресурсов областей памяти и
управления почтовыми ящиками (mailboxes) и звонками (doorbells).
