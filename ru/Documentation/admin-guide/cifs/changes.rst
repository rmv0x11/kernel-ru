=========
Изменения
=========

Сводную информацию об исправлениях и улучшениях поддержки CIFS/SMB2/SMB3
(изменениях в модуле cifs.ko) по версиям ядра (и по внутренней версии
модуля cifs) см. на https://wiki.samba.org/index.php/LinuxCIFSKernel.
Это может оказаться проще для чтения, чем разбор вывода
"git log fs/smb/client" по релизам.
