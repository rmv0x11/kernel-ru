Атрибут Intel Performance and Energy Bias в ``sysfs``
=====================================================

Значение Intel Performance and Energy Bias Hint (EPB) для заданного (логического)
CPU можно проверить или обновить через атрибут (файл) в ``sysfs``, расположенный
в :file:`/sys/devices/system/cpu/cpu<N>/power/`, где номер CPU ``<N>``
выделяется во время инициализации системы:

``energy_perf_bias``
	Показывает текущее значение EPB для CPU по скользящей шкале 0 - 15, где
	значение 0 соответствует предпочтению наивысшей производительности,
	а значение 15 соответствует максимальной экономии энергии.

	Чтобы обновить значение EPB для CPU, в этот атрибут можно произвести
	запись либо числом из приведённой выше скользящей шкалы 0 - 15, либо
	одной из строк: "performance", "balance-performance", "normal",
	"balance-power", "power", которые представляют значения, отражаемые их
	смыслом.

	Этот атрибут присутствует для всех онлайн-CPU, поддерживающих
	функциональность EPB.

Обратите внимание, что хотя интерфейс EPB к процессору определён на уровне
логического CPU, физический регистр, обеспечивающий его работу, может
совместно использоваться несколькими CPU (например, SMT-сиблингами или ядрами
в одном пакете).  По этой причине обновление значения EPB для одного CPU может
привести к изменению значений EPB для других CPU.
