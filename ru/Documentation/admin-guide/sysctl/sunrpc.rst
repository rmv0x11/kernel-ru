Этот файл содержит документацию по sysctl-файлам в
/proc/sys/sunrpc и действителен для ядра Linux версии 2.2.

Файлы в этом каталоге можно использовать для (пере)установки отладочных
флагов подсистемы SUN Remote Procedure Call (RPC) в
ядре Linux. Эта подсистема используется для NFS, KNFSD и,
возможно, ещё некоторых вещей.

Файлы в нём служат для управления отладочными флагами:
rpc_debug, nfs_debug, nfsd_debug и nlm_debug.

Эти флаги предназначены только для хакеров ядра. Для получения
дополнительной информации вам следует прочитать исходный код в net/sunrpc/.
