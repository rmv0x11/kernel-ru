Этот файл содержит документацию по sysctl-файлам, расположенным в
/proc/sys/user.

Файлы в этом каталоге можно использовать, чтобы переопределить
ограничения по умолчанию на количество namespace и других объектов,
которые имеют ограничения на одного пользователя в пределах одного
user namespace.

Основное назначение этих ограничений — остановить программы, которые
работают со сбоями и пытаются создать неоправданно большое количество
объектов, до того как этот сбой превратится в общесистемную проблему.
Подразумевается, что значения этих ограничений по умолчанию установлены
достаточно высокими, чтобы ни одна программа при нормальной работе не
упёрлась в эти ограничения.

Создание объектов, учитываемых на одного пользователя в пределах одного
user namespace, относится на счёт того пользователя в том user namespace,
который создал объект, и проверяется на соответствие ограничению на
пользователя в этом user namespace.

Создание объектов также относится на счёт всех пользователей, создавших
те user namespace, в которых происходит создание объекта (user namespace
могут быть вложенными), и проверяется на соответствие ограничениям на
пользователя в user namespace этих пользователей.

Этот рекурсивный подсчёт созданных объектов гарантирует, что создание
user namespace не позволяет пользователю обойти свои текущие ограничения.

В настоящее время в /proc/sys/user находятся следующие файлы:

max_cgroup_namespaces
=====================

  Максимальное число cgroup namespace, которое может создать любой
  пользователь в текущем user namespace.

max_ipc_namespaces
==================

  Максимальное число ipc namespace, которое может создать любой
  пользователь в текущем user namespace.

max_mnt_namespaces
==================

  Максимальное число mount namespace, которое может создать любой
  пользователь в текущем user namespace.

max_net_namespaces
==================

  Максимальное число network namespace, которое может создать любой
  пользователь в текущем user namespace.

max_pid_namespaces
==================

  Максимальное число pid namespace, которое может создать любой
  пользователь в текущем user namespace.

max_time_namespaces
===================

  Максимальное число time namespace, которое может создать любой
  пользователь в текущем user namespace.

max_user_namespaces
===================

  Максимальное число user namespace, которое может создать любой
  пользователь в текущем user namespace.

max_uts_namespaces
==================

  Максимальное число user namespace, которое может создать любой
  пользователь в текущем user namespace.
