Дополнительные возможности ноутбука LG Gram
===========================================

Автор: Matan Ziv-Av <matan@svgalib.org>


Горячие клавиши
---------------

Без этого драйвера ядро игнорирует следующие клавиши FN:

- FN-F1 (панель управления LG)   - Генерирует F15
- FN-F5 (переключение тачпада)   - Генерирует F21
- FN-F6 (режим «в самолёте»)     - Генерирует RFKILL
- FN-F9 (режим чтения)           - Генерирует F14

Остальные клавиши FN работают без необходимости в специальном драйвере.


Режим чтения
------------

Запись 0/1 в /sys/devices/platform/lg-laptop/reader_mode отключает/включает
режим чтения. В этом режиме меняются цвета экрана (уменьшается синий цвет),
и загорается светодиодный индикатор режима чтения (на клавише F9).


Блокировка FN
-------------

Запись 0/1 в /sys/devices/platform/lg-laptop/fn_lock отключает/включает
блокировку FN.


Ограничение бережной зарядки батареи
------------------------------------

Запись 80/100 в /sys/class/power_supply/CMB0/charge_control_end_threshold
задаёт максимальную ёмкость, до которой заряжается батарея. Ограничение заряда
уменьшает потерю ёмкости батареи со временем.

При загрузке ядра это значение сбрасывается в 100.


Режим вентилятора
-----------------

Запись 0/1/2 в /sys/devices/platform/lg-laptop/fan_mode задаёт режим вентилятора
Optimal/Silent/Performance соответственно.


Зарядка по USB
--------------

Запись 0/1 в /sys/devices/platform/lg-laptop/usb_charge отключает/включает
зарядку другого устройства от порта USB, пока устройство выключено.

При загрузке ядра это значение сбрасывается в 0.


Светодиоды
~~~~~~~~~~

Драйвер поддерживает два светодиодных устройства:

Подсветка клавиатуры
--------------------

Светодиодное устройство с именем kbd_led управляет подсветкой клавиатуры. Есть три
уровня освещения: выключено (0), низкий (127) и высокий (255).

Подсветкой клавиатуры также управляет комбинация клавиш FN-F8,
которая циклически переключает эти уровни.


Светодиодный индикатор тачпада
------------------------------

На клавише F5. Управляется светодиодным устройством с именем tpad_led.
