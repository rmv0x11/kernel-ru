==========================
Драйвер Alienware WMI
==========================

Kurt Borja <kuurtb@gmail.com>

Это драйвер для WMI-устройства «WMAX», которое присутствует в большинстве
игровых ноутбуков Dell и управляет различными специальными функциями.

До выпуска ноутбуков M-Series (примерно 2018 год) устройство «WMAX» управляло
базовой RGB-подсветкой, режимом глубокого сна, режимом HDMI и состоянием
усилителя.

Позднее назначение этого устройства было полностью изменено. Теперь оно в
основном работает с тепловыми профилями, мониторингом датчиков и разгоном. Этот
интерфейс называется «AWCC» и, как известно, используется OEM-приложением AWCC
для управления этими функциями.

Драйвер alienware-wmi управляет обоими интерфейсами.

Интерфейс AWCC
==============

Документация по WMI-устройству: Documentation/wmi/devices/alienware-wmi.rst

Поддерживаемые устройства
-------------------------

- Ноутбуки Alienware M-Series
- Ноутбуки Alienware X-Series
- Настольные ПК Alienware Aurora
- Ноутбуки Dell G-Series

Если вы считаете, что ваше устройство поддерживает интерфейс AWCC, но у вас
отсутствуют какие-либо из функций, описанных в этом документе, попробуйте
следующие параметры модуля alienware-wmi:

- ``force_platform_profile=1``: принудительно проверяет поддержку platform profile
- ``force_hwmon=1``: принудительно проверяет поддержку HWMON

Если модуль успешно загружается с этими параметрами, рассмотрите возможность
отправки патча, добавляющего вашу модель в ``awcc_dmi_table``, расположенную в
``drivers/platform/x86/dell/alienware-wmi-wmax.c``, либо свяжитесь с
сопровождающим для дальнейших указаний.

Состояние
---------

В настоящее время поддерживаются следующие функции:

- :ref:`Platform Profile <platform-profile>`:

  - Управление тепловым профилем

  - Переключение G-Mode

- :ref:`HWMON <hwmon>`:

  - Мониторинг датчиков

  - Ручное управление вентилятором

.. _platform-profile:

Platform Profile
----------------

Интерфейс AWCC предоставляет различные тепловые профили, определённые в прошивке.
Они предоставляются в пространство пользователя через интерфейс класса Platform
Profile. Дополнительную информацию см. в
:ref:`sysfs-class-platform-profile <abi_file_testing_sysfs_class_platform_profile>`.

Имя устройства класса platform-profile, экспортируемого этим драйвером, —
«alienware-wmi», а путь к нему можно найти с помощью:

::

 grep -l "alienware-wmi" /sys/class/platform-profile/platform-profile-*/name | sed 's|/[^/]*$||'

Если устройство поддерживает G-Mode, он также переключается при выборе профиля
``performance``.

.. note::
   Вы можете задать параметр модуля ``force_gmode``, чтобы всегда пытаться
   переключать эту функцию, не проверяя, поддерживает ли её ваша модель.

.. _hwmon:

HWMON
-----

Интерфейс AWCC также поддерживает мониторинг датчиков и ручное управление
вентилятором. Обе эти функции предоставляются в пространство пользователя через
интерфейс HWMON.

Имя устройства класса hwmon, экспортируемого этим драйвером, — «alienware_wmi»,
а путь к нему можно найти с помощью:

::

 grep -l "alienware_wmi" /sys/class/hwmon/hwmon*/name | sed 's|/[^/]*$||'

Мониторинг датчиков осуществляется через стандартный интерфейс HWMON.
Дополнительную информацию см. в
:ref:`sysfs-class-hwmon <abi_file_testing_sysfs_class_hwmon>`.

Ручное управление вентилятором, напротив, не предоставляется интерфейсом AWCC
напрямую. Вместо этого он позволяет управлять значением `boost` вентилятора. Это
значение `boost` имеет следующее приблизительное влияние на pwm вентилятора:

::

 pwm = pwm_base + (fan_boost / 255) * (pwm_max - pwm_base)

Из-за описанного выше поведения управление значением `boost` вентилятора
предоставляется в пространство пользователя через следующий пользовательский
sysfs-атрибут hwmon:

=============================== ======= =======================================
Имя				Права	Описание
=============================== ======= =======================================
fan[1-4]_boost			RW	Значение boost вентилятора.

					Целое значение от 0 до 255
=============================== ======= =======================================

.. note::
   На некоторых устройствах ручное управление вентилятором работает надёжно
   только при выборе платформенного профиля ``custom``.
