==================================
ARM DynamIQ Shared Unit (DSU) PMU
==================================

ARM DynamIQ Shared Unit объединяет одно или несколько ядер с подсистемой памяти
L3, управляющей логикой и внешними интерфейсами, образуя многоядерный кластер.
PMU позволяет подсчитывать различные события, связанные с кэшем L3, Snoop Control
Unit и т.п., с помощью независимых 32-битных счётчиков. Также он предоставляет
64-битный счётчик тактов.

К PMU можно обращаться только через системные регистры CPU, и он является общим
для ядер, подключённых к одному и тому же DSU. Как и большинство прочих uncore
PMU, DSU PMU не поддерживает события для конкретных процессов и не может
использоваться в режиме выборки (sampling).

DSU предоставляет битовую карту для подмножества реализованных событий через
аппаратные регистры. У драйвера нет способа определить, доступны ли остальные
события. Поэтому драйвер показывает только те события, которые объявлены DSU, в
каталоге "events" по пути::

  /sys/bus/event_sources/devices/arm_dsu_<N>/

Чтобы выяснить, какие события поддерживаются, и использовать сырой код события
для неперечисленных событий, пользователю следует обратиться к TRM продукта.

Драйвер также показывает CPU, подключённые к экземпляру DSU, в "associated_cpus".


пример использования::

	perf stat -a -e arm_dsu_0/cycles/
