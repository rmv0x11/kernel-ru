============================================================================================
Блок мониторинга производительности (PMU) кэша второго уровня от Qualcomm Technologies
============================================================================================

Этот драйвер поддерживает кластеры кэша L2, имеющиеся в SoC Qualcomm Technologies
Centriq. Существует несколько физических кластеров кэша L2, каждый со своим
собственным PMU. С каждым кластером связан один или несколько CPU.

Предоставляется один логический PMU L2, который агрегирует результаты от
физических PMU.

Драйвер предоставляет описание доступных событий и параметров конфигурации
в sysfs, см. /sys/bus/event_source/devices/l2cache_0.

Каталог "format" описывает формат событий.

События можно представить как двумерный массив. Каждый столбец представляет
группу событий. Всего имеется 8 групп. Одновременно может использоваться только
одна запись из каждой группы. Если задано несколько событий из одной группы,
конфликтующие события не могут подсчитываться одновременно.

События задаются в виде 0xCCG, где CC — это 2 шестнадцатеричные цифры, задающие
код (строку массива), а G задаёт группу (столбец) 0-7.

Кроме того, существует событие счётчика циклов, задаваемое значением 0xFE,
которое находится вне приведённой выше схемы.

Драйвер предоставляет атрибут sysfs "cpumask", который содержит маску,
состоящую из одного CPU на кластер, который будет использоваться для обработки всех
событий PMU на этом кластере.

Примеры использования с perf::

  perf stat -e l2cache_0/config=0x001/,l2cache_0/config=0x042/ -a sleep 1

  perf stat -e l2cache_0/config=0xfe/ -C 2 sleep 1

Драйвер не поддерживает выборку (sampling), поэтому "perf record" работать
не будет. Сеансы perf для отдельных задач (per-task) не поддерживаются.
