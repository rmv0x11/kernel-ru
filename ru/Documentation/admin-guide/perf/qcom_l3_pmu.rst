==========================================================================================
Блок мониторинга производительности (PMU) кэша L3 Qualcomm Datacenter Technologies
==========================================================================================

Этот драйвер поддерживает PMU кэша L3, имеющиеся в SoC Centriq производства
Qualcomm Datacenter Technologies. Кэш L3 на этих SoC состоит из нескольких срезов
(slices), разделяемых всеми ядрами в пределах одного сокета. Каждый срез представлен
как отдельный uncore-PMU perf с именем устройства l3cache_<socket>_<instance>.
Агрегирование по срезам возлагается на пространство пользователя.

Драйвер предоставляет описание доступных событий и параметров конфигурации в sysfs,
см. /sys/bus/event_source/devices/l3cache*. Поскольку это uncore-PMU, драйвер также
предоставляет атрибут sysfs "cpumask", который содержит маску, состоящую из одного CPU
на сокет; этот CPU будет использоваться для обработки всех событий PMU на данном сокете.

Аппаратура реализует 32-битные счётчики событий и имеет плоское 8-битное пространство
событий, доступное через атрибут формата "event". В дополнение к 32-битным физическим
счётчикам драйвер поддерживает виртуальные 64-битные аппаратные счётчики, используя
сцепление (chaining) аппаратных счётчиков. Эта возможность доступна через флаг формата
"lc" (long counter, длинный счётчик). Например::

  perf stat -e l3cache_0_0/read-miss,lc/

Поскольку это uncore-PMU, драйвер не поддерживает выборку (sampling), поэтому
"perf record" работать не будет. Сессии perf на уровне отдельных задач не поддерживаются.
