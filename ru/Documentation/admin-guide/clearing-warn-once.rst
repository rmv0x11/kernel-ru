Очистка WARN_ONCE
-----------------

WARN_ONCE / WARN_ON_ONCE / printk_once выводят сообщение только один раз.

echo 1 > /sys/kernel/debug/clear_warn_once

очищает это состояние и позволяет предупреждениям вывестись ещё раз.
Это может быть полезно после прогонов набора тестов для воспроизведения проблем.
