Драйвер TechnoTrend/Hauppauge DEC USB
=====================================

Состояние драйвера
------------------

Поддерживается:

	- DEC2000-t
	- DEC2450-t
	- DEC3000-s
	- Потоковое видео
	- Потоковое аудио
	- Секционные фильтры
	- Переключение каналов (zapping)
	- Загрузчик прошивки через hotplug

Планируется:

	- Информация о состоянии тюнера
	- Сетевой интерфейс DVB
	- Потоковая передача видео PC->DEC
	- Поддержка Conax для 2450-t

Получение прошивки
------------------
Чтобы загрузить прошивку, используйте следующие команды:

.. code-block:: none

	scripts/get_dvb_firmware dec2000t
	scripts/get_dvb_firmware dec2540t
	scripts/get_dvb_firmware dec3000s


Загрузка прошивки через hotplug
-------------------------------

Начиная с ядер 2.6, прошивка загружается в момент загрузки модуля драйвера.

Скопируйте три загруженных выше файла в каталог /usr/lib/hotplug/firmware или
/lib/firmware (в зависимости от настройки hotplug-загрузки прошивки).
