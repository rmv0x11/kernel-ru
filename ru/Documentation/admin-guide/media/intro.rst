==========
Введение
==========

Подсистема media в Linux обеспечивает поддержку нескольких различных типов
устройств:

- Аудио- и видеограбберы;
- Камеры ПК и ноутбуков;
- Сложные камеры, встречающиеся во встраиваемом оборудовании;
- Аналоговое и цифровое ТВ;
- HDMI Customer Electronics Control (CEC);
- Устройства мультитач-ввода;
- Пульты дистанционного управления;
- Медиакодеры и медиадекодеры.

В силу разнообразия устройств подсистема предоставляет несколько различных
API:

- Remote Controller API;
- HDMI CEC API;
- Video4Linux API;
- Media controller API;
- Video4Linux Request API (экспериментальный);
- Digital TV API (также известный как DVB API).
