==============================================
Драйвер камерной подсистемы Starfive (CamSS)
==============================================

Введение
--------

Этот файл документирует драйвер камерной подсистемы Starfive, присутствующей на
SoC Starfive JH7110. Драйвер расположен в drivers/staging/media/starfive/
camss.

Драйвер реализует интерфейсы V4L2, Media controller и v4l2_subdev. Поддерживается
датчик камеры, использующий интерфейс V4L2 subdev в ядре.

Драйвер успешно использовался с Gstreamer 1.18.5 и плагином v4l2src.


Оборудование камерной подсистемы Starfive
-----------------------------------------

Оборудование камерной подсистемы Starfive состоит из следующего::

                    |\         +---------------+      +-----------+
  +----------+      |  \       |               |      |           |
  |          |      |   |      |               |      |           |
  |   MIPI   |----->|   |----->|      ISP      |----->|           |
  |          |      |   |      |               |      |           |
  +----------+      |   |      |               |      |  Memory   |
                    |MUX|      +---------------+      | Interface |
  +----------+      |   |                             |           |
  |          |      |   |---------------------------->|           |
  | Parallel |----->|   |                             |           |
  |          |      |   |                             |           |
  +----------+      |  /                              |           |
                    |/                                +-----------+

- MIPI: интерфейс MIPI, принимающий данные от датчика камеры MIPI CSI-2.

- Parallel: параллельный интерфейс, принимающий данные от параллельного датчика.

- ISP: ISP, обрабатывающий необработанные данные Bayer от датчика изображения и
  формирующий кадры YUV.


Топология
---------

Граф конвейера media controller выглядит следующим образом:

.. _starfive_camss_graph:

.. kernel-figure:: starfive_camss_graph.dot
    :alt:   starfive_camss_graph.dot
    :align: center

Драйвер имеет 2 видеоустройства:

- capture_raw: устройство захвата, захватывающее данные изображения напрямую от датчика.
- capture_yuv: устройство захвата, захватывающее данные кадров YUV, обработанные
  модулем ISP

Драйвер имеет 3 субустройства:

- stf_isp: отвечает за все операции isp, выводит кадры YUV.
- cdns_csi2rx: мост CSI-2, поддерживающий до 4 линий CSI на входе и 4
  различных потока пикселей на выходе.
- imx219: датчик изображения, данные изображения передаются через MIPI CSI-2.
