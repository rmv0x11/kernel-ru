Ссылки
======

Основной сайт разработки и GIT-репозиторий драйверов цифрового
телевидения — https://linuxtv.org.

Список рассылки DVB linux-dvb размещён на vger. Подробности см. на
http://vger.kernel.org/vger-lists.html#linux-media.

Есть также несколько других старых списков, размещённых по адресу:
https://linuxtv.org/lists.php. Если они интересуют вас по историческим
причинам, обратитесь к архиву по адресу https://linuxtv.org/pipermail/linux-dvb/.

Wiki подсистемы media размещена по адресу https://linuxtv.org/wiki/.
Там вы найдёте множество информации, касающейся как разработки, так и
использования медиаплат. Пожалуйста, ознакомьтесь с ней, прежде чем задавать
вопросы новичка в списке рассылки или на IRC-каналах.

Документация по API представлена в дереве ядра. Вы можете найти её как в
формате html, так и в формате pdf, вместе с другой полезной документацией по адресу:

  - https://linuxtv.org/docs.php.

Полезные материалы также можно найти по адресу https://linuxtv.org/downloads/.

Чтобы получить прошивку, необходимую для работы некоторых драйверов,
в дереве ядра есть скрипт scripts/get_dvb_firmware.
