============================================
Raspberry Pi PiSP Camera Front End (rp1-cfe)
============================================

PiSP Camera Front End
=====================

PiSP Camera Front End (CFE) — это модуль, объединяющий приёмник CSI-2 с простым
ISP, который называется Front End (FE).

CFE содержит четыре DMA-движка и может записывать в память кадры из четырёх
отдельных потоков, полученных по CSI-2. Один из этих потоков также может быть
направлен напрямую в FE, который способен выполнять минимальную обработку
изображения, записывать в память две версии полученных кадров (например,
немасштабированную и уменьшенную) и предоставлять статистику полученных кадров.

Регистры FE задокументированы в `документе спецификации Raspberry Pi Image
Signal Processor (ISP)
<https://datasheets.raspberrypi.com/camera/raspberry-pi-image-signal-processor-specification.pdf>`_,
а пример кода для FE можно найти в `libpisp
<https://github.com/raspberrypi/libpisp>`_.

Драйвер rp1-cfe
===============

Драйвер Raspberry Pi PiSP Camera Front End (rp1-cfe) расположен в каталоге
drivers/media/platform/raspberrypi/rp1-cfe. Он использует `V4L2 API` для
регистрации ряда устройств видеозахвата и вывода, `V4L2 subdev API` для
регистрации поддустройств для приёмника CSI-2 и FE, которые соединяют
видеоустройства в единый медиа-граф, реализованный с помощью `Media Controller
(MC) API`.

Топология медиаустройств, регистрируемая драйвером `rp1-cfe`, в данном
конкретном примере подключённая к датчику imx219, выглядит следующим образом:

Медиа-граф содержит следующие узлы видеоустройств:

- rp1-cfe-csi2-ch0: устройство захвата для первого потока CSI-2
- rp1-cfe-csi2-ch1: устройство захвата для второго потока CSI-2
- rp1-cfe-csi2-ch2: устройство захвата для третьего потока CSI-2
- rp1-cfe-csi2-ch3: устройство захвата для четвёртого потока CSI-2
- rp1-cfe-fe-image0: устройство захвата для первого выхода FE
- rp1-cfe-fe-image1: устройство захвата для второго выхода FE
- rp1-cfe-fe-stats: устройство захвата для статистики FE
- rp1-cfe-fe-config: устройство вывода для конфигурации FE

rp1-cfe-csi2-chX
----------------

Устройства захвата rp1-cfe-csi2-chX — это обычные устройства захвата V4L2,
которые можно использовать для захвата видеокадров или метаданных, полученных
по CSI-2.

rp1-cfe-fe-image0, rp1-cfe-fe-image1
------------------------------------

Устройства захвата rp1-cfe-fe-image0 и rp1-cfe-fe-image1 используются для записи
обработанных кадров в память.

rp1-cfe-fe-stats
----------------

Формат буфера статистики FE определяется C-структурой
:c:type:`pisp_statistics`, а смысл каждого параметра описан в документе
`PiSP specification`.

rp1-cfe-fe-config
-----------------

Формат буфера конфигурации FE определяется C-структурой
:c:type:`pisp_fe_config`, а смысл каждого параметра описан в документе
`PiSP specification`.
