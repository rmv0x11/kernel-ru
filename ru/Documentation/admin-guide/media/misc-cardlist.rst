Тестовые драйверы
=================

Для тестирования приложений пространства пользователя (userspace) имеется ряд
виртуальных драйверов, предоставляющих тестовую функциональность и имитирующих
реальные аппаратные устройства:

=======  ======================================
Драйвер  Название
=======  ======================================
vicodec  Виртуальный драйвер кодека
vim2m    Виртуальный драйвер «память-в-память»
vimc     Виртуальный медиаконтроллер (VIMC)
vivid    Виртуальный драйвер тестирования видео
=======  ======================================
