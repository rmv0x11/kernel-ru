===================
Использование UFS
===================

mount -t ufs -o ufstype=type_of_ufs device dir


Параметры UFS
=============

ufstype=type_of_ufs
	UFS — это файловая система, широко используемая в различных операционных
	системах. Проблема заключается в различиях между реализациями. Возможности
	некоторых реализаций не документированы, поэтому трудно распознать тип ufs
	автоматически. Именно поэтому пользователь должен указать тип ufs вручную
	с помощью параметра монтирования ufstype. Возможные значения:

	old
                старый формат ufs
		значение по умолчанию, поддерживается только для чтения

	44bsd
                используется в FreeBSD, NetBSD, OpenBSD
		поддерживается для чтения и записи

	ufs2
                используется в FreeBSD 5.x
		поддерживается для чтения и записи

	5xbsd
                синоним для ufs2

	sun
                используется в SunOS (Solaris)
		поддерживается для чтения и записи

	sunx86
                используется в SunOS для Intel (Solarisx86)
		поддерживается для чтения и записи

	hp
                используется в HP-UX
		поддерживается только для чтения

	nextstep
		используется в NextStep
		поддерживается только для чтения

	nextstep-cd
		используется для NextStep CDROM (block_size == 2048)
		поддерживается только для чтения

	openstep
		используется в OpenStep
		поддерживается только для чтения
