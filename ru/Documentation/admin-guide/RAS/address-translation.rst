Трансляция адресов
==================

x86 AMD
-------

Системы AMD на базе архитектуры Zen включают компонент Data Fabric, который
управляет компоновкой физической памяти. Устройства, подключённые к Fabric,
такие как контроллеры памяти, I/O и т. д., могут не иметь полного представления
о карте физической памяти системы. Эти устройства при сообщении об ошибках
памяти могут предоставлять «нормализованный», т. е. физический адрес устройства.
Нормализованные адреса должны транслироваться в физический адрес системы, чтобы
ядро могло выполнить действия над памятью.

Библиотека трансляции адресов AMD (CONFIG_AMD_ATL) обеспечивает трансляцию для
этого случая.

Глоссарий аббревиатур, используемых при трансляции адресов для систем на базе Zen

* CCM               = Cache Coherent Moderator
* COD               = Cluster-on-Die
* COH_ST            = Coherent Station
* DF                = Data Fabric
