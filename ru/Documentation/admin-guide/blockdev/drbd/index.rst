==================================================
Distributed Replicated Block Device — DRBD
==================================================

Описание
========

  DRBD — это синхронно реплицируемое блочное устройство архитектуры
  shared-nothing. Оно спроектировано как строительный блок для кластеров
  высокой доступности (high availability) и в этом контексте является
  «drop-in»-заменой разделяемого хранилища. Упрощённо его можно
  рассматривать как сетевой RAID 1.

  Дополнительную информацию см. на https://www.drbd.org.
