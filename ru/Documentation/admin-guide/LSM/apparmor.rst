========
AppArmor
========

Что такое AppArmor?
===================

AppArmor — это расширение безопасности типа MAC для ядра Linux. Оно
реализует политику, ориентированную на задачи (task centered), при которой
«профили» (profiles) задач создаются и загружаются из пространства
пользователя. Задачи в системе, для которых не определён профиль, выполняются
в неограниченном (unconfined) состоянии, эквивалентном стандартным правам
доступа Linux DAC.

Как включить/отключить
======================

установите ``CONFIG_SECURITY_APPARMOR=y``

Если AppArmor должен быть выбран в качестве модуля безопасности по умолчанию,
то установите::

   CONFIG_DEFAULT_SECURITY_APPARMOR=y

Параметр CONFIG_LSM управляет порядком и выбором модулей LSM.
Укажите apparmor первым «основным» (major) модулем (например, AppArmor,
SELinux, Smack) в списке.

Соберите ядро

Если AppArmor не является модулем безопасности по умолчанию, его можно
включить, передав ``security=apparmor`` в командной строке ядра.

Если AppArmor является модулем безопасности по умолчанию, его можно отключить,
передав ``apparmor=0, security=XXXX`` (где ``XXXX`` — корректный модуль
безопасности) в командной строке ядра.

Чтобы AppArmor применял какие-либо ограничения помимо стандартных прав
доступа Linux DAC, в ядро из пространства пользователя должна быть загружена
политика (см. ссылки на документацию и инструменты).
