Устаревшие символы ABI
======================

Описывает интерфейсы, которые всё ещё остаются в ядре, но помечены
для удаления в некоторый момент в будущем.

Описание интерфейса документирует причину, по которой он считается
устаревшим, и когда можно ожидать его удаления.
