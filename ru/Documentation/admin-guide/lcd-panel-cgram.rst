Поддержка LCD/клавиатурных панелей на параллельном порту
==================================================================

Некоторые LCD-дисплеи позволяют определить до 8 символов, отображаемых на
ASCII-символы с 0 по 7. Управляющая последовательность для определения нового
символа — '\e[LG', за которой следует одна цифра от 0 до 7, обозначающая номер
символа, и до 8 пар шестнадцатеричных цифр, завершаемых точкой с запятой
(';'). Каждая пара цифр представляет одну строку, где единичные биты
соответствуют каждому подсвеченному пикселю, причём младший значащий бит (LSB)
находится справа. Строки нумеруются от верха символа к низу. На матрице 5x7 для
каждого символа используются только 5 младших битов из первых 7 байт. Если
строка неполная, будут переопределены только полные строки. Вот несколько
примеров::

  printf "\e[LG0010101050D1F0C04;"  => 0 = [enter]
  printf "\e[LG1040E1F0000000000;"  => 1 = [up]
  printf "\e[LG2000000001F0E0400;"  => 2 = [down]
  printf "\e[LG3040E1F001F0E0400;"  => 3 = [up-down]
  printf "\e[LG40002060E1E0E0602;"  => 4 = [left]
  printf "\e[LG500080C0E0F0E0C08;"  => 5 = [right]
  printf "\e[LG60016051516141400;"  => 6 = "IP"

  printf "\e[LG00103071F1F070301;"  => big speaker
  printf "\e[LG00002061E1E060200;"  => small speaker
