4. Программирование в пространстве пользователя
-----------------------------------------------

cfag12864bfb описывает устройство framebuffer (/dev/fbX).

Его размер составляет 1024 байта = 1 Кбайт.
Каждый бит представляет один пиксель. Если бит установлен, пиксель
включается. Если бит сброшен, пиксель выключается.

Вы можете использовать framebuffer как файл: fopen, fwrite, fclose...
Хотя LCD не будет обновлён до наступления следующего момента обновления.

Также вы можете выполнить mmap для framebuffer: open и mmap, munmap и close...
что является лучшим вариантом для большинства применений.

Смотрите samples/auxdisplay/cfag12864b-example.c
для реальной работающей полной программы в пространстве пользователя с примерами использования.
