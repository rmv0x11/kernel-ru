==============================================================
Документация драйвера extcon для ACPI-устройства Intel INT3496
==============================================================

Драйвер extcon для ACPI-устройства Intel INT3496 — это драйвер для ACPI-устройств
с acpi-id, равным INT3496, которые встречаются, например, в планшетах на базе
Intel Baytrail и Cherrytrail.

Это ACPI-устройство описывает, как ОС может считывать id-pin порта USB-otg
устройства, а также как она может опционально включать выход Vbus на порте otg и
как она может опционально управлять мультиплексированием линий данных между
USB-хостом и контроллером USB-периферии.

ACPI-устройство предоставляет эту функциональность, возвращая массив из не более
чем 3 дескрипторов gpio из своего ACPI-вызова _CRS (Current Resource Settings):

=======  =====================================================================
Index 0  Входной gpio для id-pin, он всегда присутствует и валиден
Index 1  Выходной gpio для включения выхода Vbus от устройства к порту otg,
         запишите 1, чтобы включить выход Vbus (этот дескриптор gpio может
         отсутствовать или быть невалидным)
Index 2  Выходной gpio для мультиплексирования линий данных между USB-хостом и
         контроллером USB-периферии, запишите 1, чтобы переключить на
         контроллер периферии
=======  =====================================================================

Соответствие между индексами и идентификаторами GPIO-соединений следующее

	======= =======
	id	index 0
	vbus	index 1
	mux	index 2
	======= =======
