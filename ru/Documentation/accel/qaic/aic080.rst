===============================
 Qualcomm Cloud AI 80 (AIC080)
===============================

Обзор
=====

Семейство продуктов Qualcomm Cloud AI 80/AIC080 является производным от AIC100.
Количество NSP и тактовые частоты уменьшены, чтобы вписаться в решения с
ограниченными ресурсами. PCIe Product ID — 0xa080.

Как производный продукт, наследует всю документацию AIC100.
