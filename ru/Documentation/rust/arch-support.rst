Поддержка архитектур
====================

В настоящее время компилятор Rust (``rustc``) использует LLVM для генерации
кода, что ограничивает набор поддерживаемых целевых архитектур. Кроме того,
поддержка сборки ядра с помощью LLVM/Clang различается (см.
Documentation/kbuild/llvm.rst). Эта поддержка необходима для ``bindgen``,
который использует ``libclang``.

Ниже приведена общая сводка по архитектурам, которые на данный момент
работают. Уровень поддержки соответствует значениям ``S`` в файле
``MAINTAINERS``.

=============  ===================  ==============================================
Архитектура    Уровень поддержки    Ограничения
=============  ===================  ==============================================
``arm``        Maintained           Только ARMv7 Little Endian.
``arm64``      Maintained           Только Little Endian.
``loongarch``  Maintained           \-
``riscv``      Maintained           Только ``riscv64`` и LLVM/Clang.
``um``         Maintained           \-
``x86``        Maintained           Только ``x86_64``.
=============  ===================  ==============================================
