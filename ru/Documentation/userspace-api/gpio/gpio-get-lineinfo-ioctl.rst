***********************
GPIO_GET_LINEINFO_IOCTL
***********************

.. warning::
    Этот ioctl является частью chardev_v1.rst и считается устаревшим в пользу
    gpio-v2-get-lineinfo-ioctl.rst.

Имя
===

GPIO_GET_LINEINFO_IOCTL — получить общедоступную информацию о линии.

Синопсис
========

.. c:macro:: GPIO_GET_LINEINFO_IOCTL

``int ioctl(int chip_fd, GPIO_GET_LINEINFO_IOCTL, struct gpioline_info *info)``

Аргументы
=========

``chip_fd``
    Файловый дескриптор символьного устройства GPIO, возвращённый `open()`.

``info``
    :c:type:`line_info<gpioline_info>`, который должен быть заполнен; поле
    ``offset`` задаёт линию, информацию о которой требуется получить.

Описание
========

Получить общедоступную информацию о линии.

Эта информация доступна независимо от того, используется ли линия.

.. note::
    Информация о линии не включает её значение.

    Чтобы получить доступ к значению линии, её необходимо запросить с помощью
    gpio-get-linehandle-ioctl.rst или gpio-get-lineevent-ioctl.rst.

Возвращаемое значение
=====================

В случае успеха возвращается 0, а ``info`` заполняется информацией о микросхеме.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение. Распространённые коды ошибок описаны в
error-codes.rst.
