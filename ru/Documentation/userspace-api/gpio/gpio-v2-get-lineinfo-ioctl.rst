**************************
GPIO_V2_GET_LINEINFO_IOCTL
**************************

Имя
===

GPIO_V2_GET_LINEINFO_IOCTL — получить общедоступную информацию о линии.

Синопсис
========

.. c:macro:: GPIO_V2_GET_LINEINFO_IOCTL

``int ioctl(int chip_fd, GPIO_V2_GET_LINEINFO_IOCTL, struct gpio_v2_line_info *info)``

Аргументы
=========

``chip_fd``
    Файловый дескриптор символьного устройства GPIO, возвращённый функцией `open()`.

``info``
    Структура :c:type:`line_info<gpio_v2_line_info>`, которую требуется заполнить,
    с полем ``offset``, указывающим линию, информацию о которой нужно получить.

Описание
========

Получить общедоступную информацию о линии.

Эта информация доступна независимо от того, используется линия или нет.

.. note::
    Информация о линии не включает значение линии.

    Чтобы получить доступ к значению линии, линия должна быть запрошена с помощью
    gpio-v2-get-line-ioctl.rst.

Возвращаемое значение
=====================

В случае успеха — 0, и структура ``info`` заполняется информацией о микросхеме.

В случае ошибки — -1, и переменной ``errno`` присваивается соответствующее значение.
Распространённые коды ошибок описаны в error-codes.rst.
