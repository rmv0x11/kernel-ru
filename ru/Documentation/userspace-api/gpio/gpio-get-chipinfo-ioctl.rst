***********************
GPIO_GET_CHIPINFO_IOCTL
***********************

Имя
===

GPIO_GET_CHIPINFO_IOCTL - получить публично доступную информацию о микросхеме.

Обзор
=====

.. c:macro:: GPIO_GET_CHIPINFO_IOCTL

``int ioctl(int chip_fd, GPIO_GET_CHIPINFO_IOCTL, struct gpiochip_info *info)``

Аргументы
=========

``chip_fd``
    Файловый дескриптор символьного устройства GPIO, возвращённый `open()`.

``info``
    Структура :c:type:`chip_info<gpiochip_info>`, которую необходимо заполнить.

Описание
========

Получает публично доступную информацию о конкретной микросхеме GPIO.

Возвращаемое значение
=====================

В случае успеха — 0, при этом ``info`` заполняется информацией о микросхеме.

В случае ошибки — -1, при этом переменной ``errno`` присваивается соответствующее
значение. Распространённые коды ошибок описаны в error-codes.rst.
