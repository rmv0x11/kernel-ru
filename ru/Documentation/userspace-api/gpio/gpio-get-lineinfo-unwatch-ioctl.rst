*******************************
GPIO_GET_LINEINFO_UNWATCH_IOCTL
*******************************

Название
========

GPIO_GET_LINEINFO_UNWATCH_IOCTL - Отключить отслеживание линии на предмет
изменений запрошенного состояния и сведений о конфигурации.

Синопсис
========

.. c:macro:: GPIO_GET_LINEINFO_UNWATCH_IOCTL

``int ioctl(int chip_fd, GPIO_GET_LINEINFO_UNWATCH_IOCTL, u32 *offset)``

Аргументы
=========

``chip_fd``
    Файловый дескриптор символьного устройства GPIO, возвращённый `open()`.

``offset``
    Смещение линии, которую больше не нужно отслеживать.

Описание
========

Удаляет линию из списка линий, отслеживаемых на данном ``chip_fd``.

Это операция, обратная gpio-v2-get-lineinfo-watch-ioctl.rst (v2) и
gpio-get-lineinfo-watch-ioctl.rst (v1).

Прекращение отслеживания линии, которая не отслеживается, является ошибкой (**EBUSY**).

Впервые добавлено в 5.7.

Возвращаемое значение
=====================

В случае успеха 0.

В случае ошибки -1, и переменной ``errno`` присваивается соответствующее значение.
Распространённые коды ошибок описаны в error-codes.rst.
