API eBPF для пространства пользователя
======================================

eBPF — это механизм ядра, предоставляющий изолированную (sandboxed) среду
выполнения внутри ядра Linux для расширения и инструментирования во время
работы без изменения исходного кода ядра и без загрузки модулей ядра. Программы
eBPF могут присоединяться к различным подсистемам ядра, включая сетевую
подсистему, трассировку и модули безопасности Linux (LSM).

Внутреннюю документацию ядра по eBPF см. в Documentation/bpf/index.rst.
