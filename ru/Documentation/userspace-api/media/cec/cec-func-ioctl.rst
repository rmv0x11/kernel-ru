***********
cec ioctl()
***********

Имя
===

cec-ioctl - Управление устройством cec

Синопсис
========

.. code-block:: c

    #include <sys/ioctl.h>

``int ioctl(int fd, int request, void *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``request``
    Код запроса ioctl для CEC, как определено в заголовочном файле cec.h, например
    :ref:`CEC_ADAP_G_CAPS <CEC_ADAP_G_CAPS>`.

``argp``
    Указатель на структуру, специфичную для запроса.

Описание
========

Функция :c:func:`ioctl()` управляет параметрами устройства cec. Аргумент
``fd`` должен быть открытым файловым дескриптором.

Код запроса ioctl ``request`` задаёт вызываемую функцию cec. В нём
закодировано, является ли аргумент входным, выходным или параметром
чтения/записи, а также размер аргумента ``argp`` в байтах.

Определения макросов и структур, задающих запросы ioctl для cec и
их параметры, расположены в заголовочном файле cec.h. Все запросы ioctl
для cec, их соответствующие функции и параметры описаны в
:ref:`cec-user-func`.

Возвращаемое значение
=====================

При успехе возвращается 0, при ошибке -1, и переменная ``errno``
устанавливается соответствующим образом. Общие коды ошибок описаны в
главе :ref:`Generic Error Codes <gen-errors>`.

Специфичные для запроса коды ошибок перечислены в описаниях отдельных
запросов.

Когда ioctl, принимающий выходной параметр или параметр чтения/записи,
завершается с ошибкой, этот параметр остаётся неизменным.
