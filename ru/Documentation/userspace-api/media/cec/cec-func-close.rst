***********
cec close()
***********

Имя
===

cec-close - Закрытие устройства cec

Синопсис
========

.. code-block:: c

    #include <unistd.h>

.. c:function:: int close( int fd )

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

Описание
========

Закрывает устройство cec. Ресурсы, связанные с файловым дескриптором,
освобождаются. Конфигурация устройства остаётся без изменений.

Возвращаемое значение
=====================

:c:func:`close()` возвращает 0 в случае успеха. В случае ошибки возвращается -1,
а переменной ``errno`` присваивается соответствующее значение. Возможные коды
ошибок:

``EBADF``
    ``fd`` не является корректным открытым файловым дескриптором.
