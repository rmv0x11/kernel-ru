*************
media ioctl()
*************

Имя
===

media-ioctl - Управление media-устройством

Синопсис
========

.. code-block:: c

    #include <sys/ioctl.h>

``int ioctl(int fd, int request, void *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``request``
    Код запроса media ioctl, определённый в заголовочном файле media.h,
    например MEDIA_IOC_SETUP_LINK.

``argp``
    Указатель на структуру, специфичную для запроса.

Описание
========

Функция :ref:`ioctl() <media-func-ioctl>` управляет параметрами
media-устройства. Аргумент ``fd`` должен быть открытым файловым дескриптором.

Код запроса ioctl ``request`` задаёт вызываемую media-функцию. В нём закодировано,
является ли аргумент входным, выходным параметром или параметром на чтение/запись,
а также размер аргумента ``argp`` в байтах.

Определения макросов и структур, задающих запросы media ioctl и их параметры,
расположены в заголовочном файле media.h. Все запросы media ioctl, их
соответствующие функции и параметры описаны в :ref:`media-user-func`.

Возвращаемое значение
=====================

При успешном выполнении возвращается 0, при ошибке — -1, и переменная ``errno``
устанавливается соответствующим образом. Универсальные коды ошибок описаны в
главе :ref:`Generic Error Codes <gen-errors>`.

Коды ошибок, специфичные для конкретного запроса, перечислены в описаниях
отдельных запросов.

Когда ioctl, принимающий выходной параметр или параметр на чтение/запись,
завершается с ошибкой, параметр остаётся неизменным.
