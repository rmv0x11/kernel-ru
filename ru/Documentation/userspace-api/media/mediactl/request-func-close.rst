***************
request close()
***************

Имя
===

request-close - Закрытие файлового дескриптора запроса

Синопсис
========

.. code-block:: c

    #include <unistd.h>

.. c:function:: int close( int fd )

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :ref:`MEDIA_IOC_REQUEST_ALLOC`.

Описание
========

Закрывает файловый дескриптор запроса. Ресурсы, связанные с запросом,
освобождаются после того, как все файловые дескрипторы, связанные с запросом,
будут закрыты и драйвер завершит обработку запроса.
Подробнее см. :ref:`здесь <media-request-life-time>`.

Возвращаемое значение
=====================

:c:func:`close()` возвращает 0 при успешном выполнении. При ошибке
возвращается -1, и переменной ``errno`` присваивается соответствующее значение.
Возможные коды ошибок:

EBADF
    ``fd`` не является корректным открытым файловым дескриптором.
