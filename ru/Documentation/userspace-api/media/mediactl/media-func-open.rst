************
media open()
************

Имя
===

media-open - Открытие media-устройства

Синопсис
========

.. code-block:: c

    #include <fcntl.h>

.. c:function:: int open( const char *device_name, int flags )

Аргументы
=========

``device_name``
    Открываемое устройство.

``flags``
    Флаги открытия. Режим доступа должен быть либо ``O_RDONLY``, либо ``O_RDWR``.
    Прочие флаги не оказывают влияния.

Описание
========

Для открытия media-устройства приложения вызывают :c:func:`open()` с
требуемым именем устройства. Функция не имеет побочных эффектов; конфигурация
устройства остаётся неизменной.

Когда устройство открыто в режиме только для чтения, попытки изменить его
конфигурацию приведут к ошибке, и переменной ``errno`` будет присвоено значение
EBADF.

Возвращаемое значение
=====================

В случае успеха :c:func:`open()` возвращает новый файловый дескриптор. В случае
ошибки возвращается -1, а переменной ``errno`` присваивается соответствующее
значение. Возможные коды ошибок:

EACCES
    Запрошенный доступ к файлу не разрешён.

EMFILE
    Процесс уже открыл максимально допустимое число файлов.

ENFILE
    Достигнут системный предел на общее число открытых файлов.

ENOMEM
    Недостаточно памяти ядра.

ENXIO
    Не существует устройства, соответствующего этому специальному файлу устройства.
