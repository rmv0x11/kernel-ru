*****************************
ioctl MEDIA_IOC_ENUM_ENTITIES
*****************************

Имя
===

MEDIA_IOC_ENUM_ENTITIES - Перечисление сущностей и их свойств

Синопсис
========

.. c:macro:: MEDIA_IOC_ENUM_ENTITIES

``int ioctl(int fd, MEDIA_IOC_ENUM_ENTITIES, struct media_entity_desc *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``argp``
    Указатель на struct :c:type:`media_entity_desc`.

Описание
========

Чтобы запросить атрибуты сущности, приложения устанавливают поле id
структуры :c:type:`media_entity_desc` и
вызывают ioctl MEDIA_IOC_ENUM_ENTITIES с указателем на эту
структуру. Драйвер заполняет остальную часть структуры или возвращает
код ошибки EINVAL, если id некорректен.

.. _media-ent-id-flag-next:

Сущности можно перечислять, выполняя побитовое ИЛИ id с флагом
``MEDIA_ENT_ID_FLAG_NEXT``. Драйвер вернёт информацию
о сущности с наименьшим id, строго большим запрошенного
(«следующая сущность»), либо код ошибки ``EINVAL``, если такой нет.

Идентификаторы сущностей могут быть несплошными. Приложения *не* должны пытаться
перечислять сущности, вызывая MEDIA_IOC_ENUM_ENTITIES с возрастающими
id, пока не получат ошибку.

Возвращаемое значение
=====================

При успехе возвращается 0, при ошибке -1, и переменная ``errno`` устанавливается
соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.

EINVAL
    Поле ``id`` структуры :c:type:`media_entity_desc`
    ссылается на несуществующую сущность.
