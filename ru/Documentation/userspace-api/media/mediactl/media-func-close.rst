*************
media close()
*************

Имя
===

media-close - Закрытие медиаустройства

Синопсис
========

.. code-block:: c

    #include <unistd.h>

.. c:function:: int close( int fd )

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

Описание
========

Закрывает медиаустройство. Ресурсы, связанные с файловым дескриптором,
освобождаются. Конфигурация устройства остаётся без изменений.

Возвращаемое значение
=====================

:c:func:`close()` возвращает 0 при успехе. При ошибке возвращается -1, и
``errno`` устанавливается соответствующим образом. Возможные коды ошибок:

EBADF
    ``fd`` не является корректным открытым файловым дескриптором.
