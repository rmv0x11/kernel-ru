***************
request ioctl()
***************

Имя
===

request-ioctl - Управление файловым дескриптором запроса

Синопсис
========

.. code-block:: c

    #include <sys/ioctl.h>

``int ioctl(int fd, int cmd, void *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :ref:`MEDIA_IOC_REQUEST_ALLOC`.

``cmd``
    Код команды ioctl запроса, как он определён в заголовочном файле media.h,
    например :ref:`MEDIA_REQUEST_IOC_QUEUE`.

``argp``
    Указатель на структуру, специфичную для запроса.

Описание
========

Функция :ref:`ioctl() <request-func-ioctl>` управляет параметрами запроса.
Аргумент ``fd`` должен быть открытым файловым дескриптором.

Код ioctl ``cmd`` задаёт вызываемую функцию запроса. В нём закодировано, является
ли аргумент входным, выходным или входным/выходным (read/write) параметром, а
также размер аргумента ``argp`` в байтах.

Определения макросов и структур, задающих команды ioctl запросов и их параметры,
находятся в заголовочном файле media.h. Все команды ioctl запросов, их
соответствующие функции и параметры описаны в :ref:`media-user-func`.

Возвращаемое значение
=====================

В случае успеха возвращается 0, в случае ошибки -1, и переменная ``errno``
устанавливается соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.

Специфичные для команд коды ошибок перечислены в описаниях отдельных команд.

Когда ioctl, принимающий выходной или входной/выходной (read/write) параметр,
завершается с ошибкой, параметр остаётся неизменным.
