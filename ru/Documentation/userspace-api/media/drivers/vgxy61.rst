Драйвер датчика камеры ST VGXY61
================================

Драйвер ST VGXY61 реализует следующие элементы управления:

``V4L2_CID_HDR_SENSOR_MODE``
-------------------------------
    Изменяет режим HDR датчика. HDR-изображение получается путём слияния двух
    снимков одной и той же сцены, сделанных с двумя разными периодами экспозиции.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - HDR linearize
      - Блок слияния выдаёт снимок с длинной экспозицией до тех пор, пока он не
        насыщается.
    * - HDR subtraction
      - Это предполагает вычитание кадра с короткой экспозицией из кадра с
        длинной экспозицией.
    * - No HDR
      - Этот режим используется для экспозиций со стандартным динамическим
        диапазоном (SDR).
