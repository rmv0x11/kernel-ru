Драйвер ISP THine THP7312
=========================

Драйвер THP7312 реализует следующие специфичные для драйвера элементы управления:

``V4L2_CID_THP7312_LOW_LIGHT_COMPENSATION``
    Включение/отключение автоматической подстройки частоты кадров в зависимости
    от условий освещённости при включённой автоэкспозиции.

``V4L2_CID_THP7312_AUTO_FOCUS_METHOD``
    Задаёт метод автофокусировки. Действует только при включённой автофокусировке.

    .. flat-table::
        :header-rows:  0
        :stub-columns: 0
        :widths:       1 4

        * - ``0``
          - Автофокусировка на основе контраста
        * - ``1``
          - PDAF
        * - ``2``
          - Гибрид автофокусировки на основе контраста и PDAF

    Поддерживаемые для этого элемента управления значения зависят от модуля
    датчика камеры, подключённого к THP7312. Если модуль не имеет привода объектива
    для фокусировки, этот элемент управления не будет предоставлен драйвером THP7312.
    Если модуль имеет управляемый объектив фокусировки, но датчик не поддерживает
    PDAF, допустимым будет только значение автофокусировки на основе контраста.
    В противном случае будут поддерживаться все значения этого элемента управления.

``V4L2_CID_THP7312_NOISE_REDUCTION_AUTO``
    Включение/отключение автоматического шумоподавления.

``V4L2_CID_THP7312_NOISE_REDUCTION_ABSOLUTE``
    Задаёт силу шумоподавления, где 0 — самое слабое, а 10 — самое сильное.
