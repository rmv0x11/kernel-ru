Драйвер тюнера Maxim Integrated MAX2175 «RF to bits»
====================================================

Драйвер MAX2175 реализует следующие специфичные для драйвера элементы управления:

``V4L2_CID_MAX2175_I2S_ENABLE``
-------------------------------
    Включение/отключение вывода I2S тюнера. Это приватный элемент управления,
    доступ к которому возможен только через интерфейс subdev.
    Дополнительные сведения см. в Documentation/driver-api/media/v4l2-controls.rst.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - ``(0)``
      - Вывод I2S отключён.
    * - ``(1)``
      - Вывод I2S включён.

``V4L2_CID_MAX2175_HSLS``
-------------------------
    Управление режимом high-side/low-side (HSLS) тюнера для заданного диапазона.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - ``(0)``
      - Положение частоты LO ниже желаемой частоты.
    * - ``(1)``
      - Положение частоты LO выше желаемой частоты.

``V4L2_CID_MAX2175_RX_MODE (menu)``
-----------------------------------
    Режим Rx управляет рядом предустановленных параметров тюнера, таких как
    тактовая частота дискретизации (sck), частота дискретизации и т. д. Эти
    несколько настроек объединены под одной меткой, называемой в техническом
    описании режимом Rx. В приведённом ниже списке показаны поддерживаемые режимы
    с кратким описанием.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - ``"Europe modes"``
    * - ``"FM 1.2" (0)``
      - Настраивает диапазон FM с частотой дискретизации 0,512 миллиона
        отсчётов/сек при sck 10,24 МГц.
    * - ``"DAB 1.2" (1)``
      - Настраивает диапазон VHF с частотой дискретизации 2,048 миллиона
        отсчётов/сек при sck 32,768 МГц.

    * - ``"North America modes"``
    * - ``"FM 1.0" (0)``
      - Настраивает диапазон FM с частотой дискретизации 0,7441875 миллиона
        отсчётов/сек при sck 14,88375 МГц.
    * - ``"DAB 1.2" (1)``
      - Настраивает диапазон FM с частотой дискретизации 0,372 миллиона
        отсчётов/сек при sck 7,441875 МГц.
