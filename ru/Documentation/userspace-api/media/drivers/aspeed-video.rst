Драйвер видео ASPEED
====================

Видеодвижок ASPEED (ASPEED Video Engine), присутствующий в SoC AST2400/2500/2600,
поддерживает высокопроизводительное сжатие видео с широким диапазоном параметров
качества видео и коэффициента сжатия. Используемый алгоритм сжатия — это
модифицированный алгоритм JPEG.

В этом IP-блоке есть 2 типа сжатия.

* Стандартный режим JPEG JFIF: для сжатия одиночных кадров и для нужд управления.
* Проприетарный режим ASPEED: для многокадрового и дифференциального сжатия.
  Поддерживает схему двухпроходного (высококачественного) сжатия видео (патент
  ASPEED в стадии оформления). Обеспечивает визуально без потерь качество сжатия
  видео либо позволяет снизить среднюю нагрузку на сеть в приложениях KVM
  в пределах интрасети.

VIDIOC_S_FMT можно использовать для выбора желаемого формата. V4L2_PIX_FMT_JPEG
обозначает стандартный режим JPEG JFIF; V4L2_PIX_FMT_AJPG обозначает проприетарный
режим ASPEED.

Более подробные сведения об операциях аппаратного обеспечения видео ASPEED можно
найти в *главе 6.2.16 KVM Video Driver* документа SDK_User_Guide, который доступен
на `github <https://github.com/AspeedTech-BMC/openbmc/releases/>`__.

Драйвер видео ASPEED реализует следующий специфичный для драйвера элемент управления:

``V4L2_CID_ASPEED_HQ_MODE``
---------------------------
    Включает/отключает режим высокого качества (High quality mode) ASPEED. Это
    приватный элемент управления, который можно использовать для включения высокого
    качества в проприетарном режиме aspeed.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - ``(0)``
      - Режим HQ ASPEED отключён.
    * - ``(1)``
      - Режим HQ ASPEED включён.

``V4L2_CID_ASPEED_HQ_JPEG_QUALITY``
-----------------------------------
    Определяет качество режима высокого качества (High quality mode) ASPEED. Это
    приватный элемент управления, который можно использовать для выбора качества
    сжатия, если включён режим высокого качества. Чем выше значение, тем лучше
    качество и больше размер.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 4

    * - ``(1)``
      - минимум
    * - ``(12)``
      - максимум
    * - ``(1)``
      - шаг
    * - ``(1)``
      - по умолчанию
