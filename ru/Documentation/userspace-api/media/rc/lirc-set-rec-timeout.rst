***************************************************
ioctl LIRC_GET_REC_TIMEOUT and LIRC_SET_REC_TIMEOUT
***************************************************

Имя
===

LIRC_GET_REC_TIMEOUT/LIRC_SET_REC_TIMEOUT - получить/установить целочисленное значение тайм-аута бездействия ИК.

Синопсис
========

.. c:macro:: LIRC_GET_REC_TIMEOUT

``int ioctl(int fd, LIRC_GET_REC_TIMEOUT, __u32 *timeout)``

.. c:macro:: LIRC_SET_REC_TIMEOUT

``int ioctl(int fd, LIRC_SET_REC_TIMEOUT, __u32 *timeout)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый функцией open().

``timeout``
    Тайм-аут в микросекундах.

Описание
========

Получает и устанавливает целочисленное значение тайм-аута бездействия ИК.

Если это поддерживается оборудованием, установка значения в 0 отключает все
аппаратные тайм-ауты, и данные должны сообщаться как можно быстрее. Если точное
значение установить невозможно, то должно быть установлено следующее возможное
значение, *большее* заданного.

.. note::

   Диапазон поддерживаемых тайм-аутов задаётся :ref:`LIRC_GET_MIN_TIMEOUT`.

Возвращаемое значение
=====================

В случае успеха возвращается 0, в случае ошибки -1, а переменная ``errno``
устанавливается соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
