.. _lirc_set_transmitter_mask:

*******************************
ioctl LIRC_SET_TRANSMITTER_MASK
*******************************

Имя
===

LIRC_SET_TRANSMITTER_MASK - Включает отправку кодов на заданном наборе передатчиков

Синопсис
========

.. c:macro:: LIRC_SET_TRANSMITTER_MASK

``int ioctl(int fd, LIRC_SET_TRANSMITTER_MASK, __u32 *mask)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый open().

``mask``
    Маска с каналами для включения передачи (tx). Канал 0 — это наименее значимый бит.

Описание
========

Некоторые IR TX устройства имеют несколько выходных каналов; в этом случае
:ref:`LIRC_CAN_SET_TRANSMITTER_MASK <LIRC-CAN-SET-TRANSMITTER-MASK>`
возвращается через :ref:`LIRC_GET_FEATURES`, и данный ioctl задаёт, какие каналы
будут отправлять IR-коды.

Данный ioctl включает заданный набор передатчиков. Первый передатчик
кодируется наименее значимым битом и так далее.

Если задана недопустимая битовая маска, то есть установлен бит, хотя устройство
не имеет такого количества передатчиков, то данный ioctl возвращает количество
доступных передатчиков и ничего более не делает.

Возвращаемое значение
=====================

В случае успеха возвращается 0, в случае ошибки -1, и переменная ``errno``
устанавливается соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
