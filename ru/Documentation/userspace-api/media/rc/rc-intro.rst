.. _Remote_controllers_Intro:

********
Введение
********

В настоящее время большинство аналоговых и цифровых устройств имеют
инфракрасный вход для пультов дистанционного управления. У каждого
производителя свой тип управления. Нередко один и тот же производитель
поставляет разные типы пультов в зависимости от устройства.

Интерфейс пульта дистанционного управления отображается как обычный
интерфейс evdev/input, точно так же, как клавиатура или мышь. Поэтому он
использует все ioctl, уже определённые для любых других устройств ввода.

Однако пульты дистанционного управления более гибки, чем обычное
устройство ввода, поскольку ИК-приёмник (и/или передатчик) может
использоваться совместно с широким спектром различных ИК-пультов.

Чтобы обеспечить гибкость, подсистема пультов дистанционного управления
позволяет управлять специфичными для RC атрибутами через
:ref:`узлы класса sysfs <remote_controllers_sysfs_nodes>`.
