********************************
ioctl LIRC_SET_WIDEBAND_RECEIVER
********************************

Имя
===

LIRC_SET_WIDEBAND_RECEIVER - включить широкополосный приёмник.

Синопсис
========

.. c:macro:: LIRC_SET_WIDEBAND_RECEIVER

``int ioctl(int fd, LIRC_SET_WIDEBAND_RECEIVER, __u32 *enable)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый open().

``enable``
    enable = 1 означает включить широкополосный приёмник, enable = 0 означает
    отключить широкополосный приёмник.

Описание
========

Некоторые приёмники оснащены специальным широкополосным приёмником, который
предназначен для считывания сигнала существующего пульта. Данный ioctl
позволяет включать или отключать его.

Это может быть полезно для приёмников, которые в остальном имеют узкополосный
приёмник, не позволяющий использовать их с некоторыми пультами. Широкополосный
приёмник также может быть более точным. С другой стороны, его недостатком обычно
является уменьшенная дальность приёма.

.. note::

    Широкополосный приёмник может быть неявно включён, если вы включите отчёты
    о несущей. В этом случае он будет отключён, как только вы отключите отчёты
    о несущей. Попытка отключить широкополосный приёмник, пока активны отчёты
    о несущей, не приведёт ни к какому результату.

Возвращаемое значение
=====================

В случае успеха возвращается 0, при ошибке -1, и переменная ``errno``
устанавливается соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
