******************************
ioctl LIRC_SET_SEND_DUTY_CYCLE
******************************

Имя
===

LIRC_SET_SEND_DUTY_CYCLE - Задаёт скважность (duty cycle) несущего сигнала
для передачи ИК.

Обзор
=====

.. c:macro:: LIRC_SET_SEND_DUTY_CYCLE

``int ioctl(int fd, LIRC_SET_SEND_DUTY_CYCLE, __u32 *duty_cycle)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый open().

``duty_cycle``
    Скважность (duty cycle), описывающая ширину импульса в процентах (от 1 до 99)
    от полного цикла. Значения 0 и 100 зарезервированы.

Описание
========

Получает/задаёт скважность (duty cycle) несущего сигнала для передачи ИК.

В настоящее время для значений 0 и 100 не определено никакого особого смысла,
но в будущем они могут использоваться для отключения генерации несущей, поэтому
эти значения следует считать зарезервированными.

Возвращаемое значение
=====================

При успехе возвращается 0, при ошибке -1, и переменной ``errno`` присваивается
соответствующее значение. Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
