.. _Remote_controllers_table_change:

*******************************************************
Изменение раскладок по умолчанию для пультов управления
*******************************************************

Интерфейс событий предоставляет два ioctl для использования с устройством
/dev/input/event, что позволяет изменять раскладку клавиш по умолчанию.

Эта программа демонстрирует, как заменить таблицы раскладок клавиш.
