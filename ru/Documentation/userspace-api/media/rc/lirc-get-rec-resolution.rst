*****************************
ioctl LIRC_GET_REC_RESOLUTION
*****************************

Имя
===

LIRC_GET_REC_RESOLUTION - Получить значение разрешения приёма в микросекундах.

Синопсис
========

.. c:macro:: LIRC_GET_REC_RESOLUTION

``int ioctl(int fd, LIRC_GET_REC_RESOLUTION, __u32 *microseconds)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый open().

``microseconds``
    Разрешение в микросекундах.

Описание
========

Некоторые приёмники имеют максимальное разрешение, которое определяется
ограничениями внутренней частоты дискретизации или формата данных. Например,
часто сигналы могут сообщаться только с шагом 50 микросекунд.

Этот ioctl возвращает целочисленное значение с таким разрешением, которое может
использоваться приложениями пространства пользователя, такими как lircd, для
автоматической подстройки значения допуска.

Возвращаемое значение
=====================

При успехе возвращается 0, при ошибке -1, а переменная ``errno`` устанавливается
соответствующим образом. Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
