**************************
ioctl LIRC_SET_REC_CARRIER
**************************

Имя
===

LIRC_SET_REC_CARRIER - Задаёт несущую частоту, используемую для модуляции при приёме ИК-сигнала.

Синопсис
========

.. c:macro:: LIRC_SET_REC_CARRIER

``int ioctl(int fd, LIRC_SET_REC_CARRIER, __u32 *frequency)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый функцией open().

``frequency``
    Частота несущей, модулирующей данные PWM, в Гц.

Описание
========

Задаёт несущую частоту приёма, используемую для модуляции импульсов и пауз ИК PWM.

.. note::

   При вызове совместно с :ref:`LIRC_SET_REC_CARRIER_RANGE` этот ioctl
   задаёт верхнюю границу частоты, которая будет распознаваться устройством.

Возвращаемое значение
=====================

В случае успеха возвращается 0, в случае ошибки -1, и переменной ``errno``
присваивается соответствующее значение. Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
