==============
DMX_REMOVE_PID
==============

Имя
---

DMX_REMOVE_PID

Синтаксис
---------

.. c:macro:: DMX_REMOVE_PID

``int ioctl(fd, DMX_REMOVE_PID, __u16 *pid)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``pid``
    PID PES-фильтра, который требуется удалить.

Описание
--------

Этот вызов ioctl позволяет удалить PID, когда на фильтре транспортного
потока задано несколько PID — например, на фильтре, ранее настроенном с
выводом, равным :c:type:`DMX_OUT_TSDEMUX_TAP <dmx_output>`, и созданном
посредством :ref:`DMX_SET_PES_FILTER` либо :ref:`DMX_ADD_PID`.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
