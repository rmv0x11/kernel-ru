*****************************************
Цифровое ТВ: close() для frontend
*****************************************

Имя
===

fe-close - Закрытие устройства frontend

Синопсис
========

.. code-block:: c

    #include <unistd.h>

.. c:function:: int close( int fd )

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

Описание
========

Этот системный вызов закрывает ранее открытое устройство frontend. После
закрытия устройства frontend соответствующее ему оборудование может быть
автоматически выключено.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
