************
DVB munmap()
************

Имя
===

dmx-munmap - Отмена отображения памяти устройства

.. warning:: Этот API всё ещё является экспериментальным.

Синопсис
========

.. code-block:: c

    #include <unistd.h>
    #include <sys/mman.h>

.. c:function:: int munmap( void *start, size_t length )

Аргументы
=========

``start``
    Адрес отображённого буфера, возвращённый функцией
    :c:func:`mmap()`.

``length``
    Длина отображённого буфера. Это значение должно совпадать с тем, которое
    было передано функции :c:func:`mmap()`.

Описание
========

Отменяет отображение буфера, ранее отображённого функцией :c:func:`mmap()`,
и освобождает его, если это возможно.

Возвращаемое значение
=====================

В случае успеха :c:func:`munmap()` возвращает 0, в случае неудачи -1, и
переменной ``errno`` присваивается соответствующее значение:

EINVAL
    Значение ``start`` или ``length`` некорректно, либо ещё не было
    отображено ни одного буфера.
