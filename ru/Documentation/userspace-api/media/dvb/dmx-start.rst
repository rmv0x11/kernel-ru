=========
DMX_START
=========

Имя
---

DMX_START

Обзор
-----

.. c:macro:: DMX_START

``int ioctl(int fd, DMX_START)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

Описание
--------

Этот вызов ioctl используется для запуска самой операции фильтрации,
заданной через вызовы ioctl :ref:`DMX_SET_FILTER` или :ref:`DMX_SET_PES_FILTER`.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    -  .. row 1

       -  ``EINVAL``

       -  Недопустимый аргумент, т.е. не предоставлены параметры фильтрации

    -  .. row 2

       -  ``EBUSY``

       -  Этот код ошибки указывает на наличие конфликтующих запросов.

	  Убедитесь, что эти фильтры остановлены, прежде чем запускать данный
	  фильтр.
