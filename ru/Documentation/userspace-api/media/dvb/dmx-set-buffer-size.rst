===================
DMX_SET_BUFFER_SIZE
===================

Имя
---

DMX_SET_BUFFER_SIZE

Синопсис
--------

.. c:macro:: DMX_SET_BUFFER_SIZE

``int ioctl(int fd, DMX_SET_BUFFER_SIZE, unsigned long size)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``size``
    Размер типа unsigned long

Описание
--------

Этот вызов ioctl используется для установки размера кольцевого буфера,
применяемого для отфильтрованных данных. Размер по умолчанию равен двум
секциям максимального размера, то есть если данная функция не вызывается,
то будет использоваться буфер размером ``2 * 4096`` байт.

Возвращаемое значение
---------------------

При успешном выполнении возвращается 0.

При ошибке возвращается -1, и переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
