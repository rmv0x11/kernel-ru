================
DMX_GET_PES_PIDS
================

Имя
---

DMX_GET_PES_PIDS

Обзор
-----

.. c:macro:: DMX_GET_PES_PIDS

``int ioctl(fd, DMX_GET_PES_PIDS, __u16 pids[5])``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``pids``
    Массив, используемый для хранения 5 идентификаторов программ (Program ID).

Описание
--------

Данный ioctl позволяет запросить у устройства DVB первый PID, используемый
аудио-, видео-, телетекст-, субтитровыми программами и программой PCR в заданном
сервисе. Они хранятся следующим образом:

=======================	========	=======================================
элемент PID		позиция		содержимое
=======================	========	=======================================
pids[DMX_PES_AUDIO]	0		первый аудио-PID
pids[DMX_PES_VIDEO]	1		первый видео-PID
pids[DMX_PES_TELETEXT]	2		первый PID телетекста
pids[DMX_PES_SUBTITLE]	3		первый PID субтитров
pids[DMX_PES_PCR]	4		первый PID Program Clock Reference
=======================	========	=======================================

.. note::

	Значение, равное 0xffff, означает, что данный PID не был заполнен
	ядром.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
