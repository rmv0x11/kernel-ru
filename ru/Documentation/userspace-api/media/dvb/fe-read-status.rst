********************
ioctl FE_READ_STATUS
********************

Имя
===

FE_READ_STATUS - возвращает информацию о состоянии входного каскада (front-end). Для этого вызова требуется только доступ к устройству на чтение

Синопсис
========

.. c:macro:: FE_READ_STATUS

``int ioctl(int fd, FE_READ_STATUS, unsigned int *status)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``status``
    указатель на целое число — битовую маску, заполняемую значениями, определёнными
    в перечислении :c:type:`fe_status`.

Описание
========

Все цифровые ТВ-устройства входного каскада поддерживают ioctl ``FE_READ_STATUS``. Он
используется для проверки состояния захвата (locking) входного каскада после
настройки. Этот ioctl принимает указатель на целое число, в которое будет
записано состояние.

.. note::

   Фактический размер status составляет sizeof(enum fe_status), который
   варьируется в зависимости от архитектуры. В будущем это нужно будет
   исправить.

int fe_status
=============

Параметр fe_status используется для указания текущего состояния и/или
изменений состояния аппаратуры входного каскада. Он формируется с использованием значений
перечисления :c:type:`fe_status` в виде битовой маски

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
