***********************
FE_READ_SIGNAL_STRENGTH
***********************

Имя
===

FE_READ_SIGNAL_STRENGTH

.. attention:: Этот ioctl является устаревшим.

Краткое описание
================

.. c:macro:: FE_READ_SIGNAL_STRENGTH

``int ioctl(int fd, FE_READ_SIGNAL_STRENGTH, uint16_t *strength)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``strength``
    Значение силы сигнала сохраняется в \*strength.

Описание
========

Этот вызов ioctl возвращает значение силы сигнала для сигнала,
принимаемого фронтендом в данный момент. Для этой команды достаточно
доступа к устройству только на чтение.

Возвращаемое значение
=====================

При успешном выполнении возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
