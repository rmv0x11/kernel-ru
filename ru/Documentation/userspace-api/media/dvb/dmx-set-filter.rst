.. _DMX_SET_FILTER:

==============
DMX_SET_FILTER
==============

Имя
---

DMX_SET_FILTER

Обзор
-----

.. c:macro:: DMX_SET_FILTER

``int ioctl(int fd, DMX_SET_FILTER, struct dmx_sct_filter_params *params)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``params``

    Указатель на структуру, содержащую параметры фильтра.

Описание
--------

Этот вызов ioctl настраивает фильтр в соответствии с предоставленными
параметрами фильтра и маски. Может быть задан тайм-аут, указывающий число
секунд ожидания загрузки секции. Значение 0 означает, что тайм-аут
применяться не должен. Наконец, имеется поле флагов, в котором можно
указать, следует ли проверять CRC секции, должен ли фильтр быть фильтром
типа «one-shot», то есть должна ли операция фильтрации останавливаться
после приёма первой секции, а также должна ли операция фильтрации
запускаться немедленно (без ожидания вызова ioctl
:ref:`DMX_START`). Если ранее был настроен фильтр, этот
фильтр будет отменён, а приёмный буфер будет очищен.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
