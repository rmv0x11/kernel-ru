FE_DISHNETWORK_SEND_LEGACY_CMD
******************************

Имя
===

FE_DISHNETWORK_SEND_LEGACY_CMD

Обзор
=====

.. c:macro:: FE_DISHNETWORK_SEND_LEGACY_CMD

``int ioctl(int fd, FE_DISHNETWORK_SEND_LEGACY_CMD, unsigned long cmd)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``cmd``
    Отправляет указанную необработанную команду cmd на антенну (dish) через DISEqC.

Описание
========

.. warning::
   Это очень малоизвестная устаревшая команда, используемая только в
   драйвере stv0299. Не должна использоваться в более новых драйверах.

Она предоставляет нестандартный метод выбора напряжения Diseqc на
frontend для устаревших переключателей Dish Network.

Поскольку поддержка этого ioctl была добавлена в 2004 году, это означает,
что такие антенны уже были устаревшими в 2004 году.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
