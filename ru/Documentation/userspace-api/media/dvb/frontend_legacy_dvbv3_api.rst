.. _frontend_legacy_dvbv3_api:

************************************************************
Устаревший API фронтенда цифрового ТВ (он же DVBv3)
************************************************************

Использование этого API не рекомендуется, так как он поддерживает не все
стандарты цифрового ТВ, не обеспечивает качественного измерения статистики
и предоставляет неполную информацию. Он сохраняется только для поддержки
устаревших приложений.
