******************************
ioctl FE_DISEQC_RESET_OVERLOAD
******************************

Имя
===

FE_DISEQC_RESET_OVERLOAD - Восстанавливает питание антенной подсистемы, если оно было отключено из-за перегрузки по питанию.

Синопсис
========

.. c:macro:: FE_DISEQC_RESET_OVERLOAD

``int ioctl(int fd, FE_DISEQC_RESET_OVERLOAD, NULL)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

Описание
========

Если шина была автоматически обесточена из-за перегрузки по питанию,
этот вызов ioctl восстанавливает питание шины. Вызов требует
доступа к устройству на чтение/запись. Этот вызов не имеет эффекта, если устройство
было обесточено вручную. Не все адаптеры цифрового ТВ поддерживают данный вызов ioctl.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
