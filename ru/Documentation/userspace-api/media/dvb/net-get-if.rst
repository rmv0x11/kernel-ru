****************
ioctl NET_GET_IF
****************

Имя
===

NET_GET_IF - Чтение конфигурационных данных интерфейса, созданного через - :ref:`NET_ADD_IF <net>`.

Синопсис
========

.. c:macro:: NET_GET_IF

``int ioctl(int fd, NET_GET_IF, struct dvb_net_if *net_if)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``net_if``
    указатель на структуру :c:type:`dvb_net_if`

Описание
========

ioctl NET_GET_IF использует номер интерфейса, заданный полем
:c:type:`dvb_net_if`::ifnum, и заполняет содержимое
структуры :c:type:`dvb_net_if` идентификатором пакета (packet ID) и
типом инкапсуляции, используемыми на этом интерфейсе. Если интерфейс ещё не был
создан с помощью :ref:`NET_ADD_IF <net>`, будет возвращено -1, а в
``errno`` будет записан код ошибки ``EINVAL``.

Возвращаемое значение
=====================

В случае успеха возвращается 0, и :c:type:`ca_slot_info` заполняется.

В случае ошибки возвращается -1, и переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
