.. _FE_GET_PROPERTY:

**************************************
ioctl FE_SET_PROPERTY, FE_GET_PROPERTY
**************************************

Имя
===

FE_SET_PROPERTY - FE_GET_PROPERTY - FE_SET_PROPERTY устанавливает одно или несколько свойств фронтенда. - FE_GET_PROPERTY возвращает одно или несколько свойств фронтенда.

Обзор
=====

.. c:macro:: FE_GET_PROPERTY

``int ioctl(int fd, FE_GET_PROPERTY, struct dtv_properties *argp)``

.. c:macro:: FE_SET_PROPERTY

``int ioctl(int fd, FE_SET_PROPERTY, struct dtv_properties *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``argp``
    Указатель на struct :c:type:`dtv_properties`.

Описание
========

Все фронтенд-устройства цифрового ТВ поддерживают ioctl-вызовы ``FE_SET_PROPERTY`` и
``FE_GET_PROPERTY``. Поддерживаемые свойства и статистика
зависят от системы доставки и от устройства:

-  ``FE_SET_PROPERTY:``

   -  Этот ioctl используется для установки одного или нескольких свойств фронтенда.

   -  Это базовая команда, чтобы дать фронтенду указание настроиться на
      некоторую частоту и начать декодирование цифрового ТВ-сигнала.

   -  Этот вызов требует доступа к устройству на чтение/запись.

.. note::

   При возврате значения не обновляются для отражения фактически
   использованных параметров. Если нужны фактические параметры, требуется явный
   вызов ``FE_GET_PROPERTY``.

-  ``FE_GET_PROPERTY:``

   -  Этот ioctl используется для получения свойств и статистики от
      фронтенда.

   -  Никакие свойства не изменяются, и статистика не сбрасывается.

   -  Этот вызов требует доступа к устройству только на чтение.

Возвращаемое значение
=====================

При успехе возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
