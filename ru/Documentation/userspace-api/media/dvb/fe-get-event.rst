.. _FE_GET_EVENT:

************
FE_GET_EVENT
************

Имя
===

FE_GET_EVENT

.. attention:: Этот ioctl является устаревшим.

Обзор
=====

.. c:macro:: FE_GET_EVENT

``int ioctl(int fd, FE_GET_EVENT, struct dvb_frontend_event *ev)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``ev``
    Указывает на место, в котором должно быть сохранено событие, если оно есть.

Описание
========

Этот вызов ioctl возвращает событие фронтенда, если оно доступно. Если событие
недоступно, поведение зависит от того, находится ли устройство в блокирующем или
неблокирующем режиме. В последнем случае вызов сразу завершается неудачей, при этом
errno устанавливается в ``EWOULDBLOCK``. В первом случае вызов блокируется до тех пор,
пока событие не станет доступным.

Возвращаемое значение
=====================

При успешном выполнении возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    -  .. row 1

       -  ``EWOULDBLOCK``

       -  Нет ожидающих событий, и устройство находится в неблокирующем режиме.

    -  .. row 2

       -  ``EOVERFLOW``

       -  Переполнение очереди событий — одно или несколько событий было потеряно.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
