****************
ioctl NET_ADD_IF
****************

Имя
===

NET_ADD_IF - Создаёт новый сетевой интерфейс для заданного Packet ID.

Синопсис
========

.. c:macro:: NET_ADD_IF

``int ioctl(int fd, NET_ADD_IF, struct dvb_net_if *net_if)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``net_if``
    указатель на struct :c:type:`dvb_net_if`

Описание
========

Системный вызов ioctl NET_ADD_IF выбирает Packet ID (PID), который
содержит TCP/IP-трафик, тип используемой инкапсуляции (MPE или
ULE) и номер интерфейса для нового создаваемого интерфейса. Когда
системный вызов успешно завершается, создаётся новый виртуальный сетевой
интерфейс.

Поле struct :c:type:`dvb_net_if`::ifnum будет
заполнено номером созданного интерфейса.

Возвращаемое значение
=====================

В случае успеха возвращается 0, и :c:type:`ca_slot_info` заполняется.

В случае ошибки возвращается -1, и переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
