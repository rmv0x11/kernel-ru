###############################
Часть II — API цифрового ТВ
###############################

.. note::

   Этот API также известен как Linux **DVB API**.

   Изначально он был написан для поддержки европейского стандарта цифрового
   ТВ (DVB), а позднее расширен для поддержки всех стандартов цифрового ТВ.

   Чтобы избежать путаницы, в данном документе было решено называть его, а
   также связанное с ним оборудование, **Digital TV** (цифровое ТВ).

   Слово **DVB** зарезервировано для использования в значении:

     - версии API цифрового ТВ
       (например, DVB API версии 3 или DVB API версии 5);
     - типов данных цифрового ТВ (enum, struct, define и т. п.);
     - узлов устройств цифрового ТВ (``/dev/dvb/...``);
     - европейского стандарта DVB.
