====================================
read() демультиплексора цифрового ТВ
====================================

Имя
---

read() демультиплексора цифрового ТВ

Синопсис
--------

.. c:function:: size_t read(int fd, void *buf, size_t count)

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

 ``buf``
   Заполняемый буфер

``count``
   Максимальное число байтов для чтения

Описание
--------

Этот системный вызов возвращает отфильтрованные данные, которые могут быть
данными секции или Packetized Elementary Stream (PES). Отфильтрованные данные
передаются из внутреннего кольцевого буфера драйвера в ``buf``. Максимальный
объём передаваемых данных определяется значением count.

.. note::

   если для секционного фильтра, созданного с установленным флагом
   :c:type:`DMX_CHECK_CRC <dmx_sct_filter_params>`,
   данные не проходят проверку CRC, они молча игнорируются.

Возвращаемое значение
---------------------

При успехе возвращается 0.

При ошибке возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

.. tabularcolumns:: |p{2.5cm}|p{15.0cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths: 1 16

    -  -  ``EWOULDBLOCK``
       -  Нет данных для возврата, и был указан ``O_NONBLOCK``.

    -  -  ``EOVERFLOW``
       -  Отфильтрованные данные не были своевременно прочитаны из буфера,
	  в результате чего непрочитанные данные были потеряны. Буфер сбрасывается.

    -  -  ``ETIMEDOUT``
       -  Секция не была загружена в течение указанного периода тайм-аута.
          О том, как задать тайм-аут, см. ioctl :ref:`DMX_SET_FILTER`.

    -  -  ``EFAULT``
       -  Драйверу не удалось выполнить запись в буфер вызывающего из-за
          некорректного указателя \*buf.

Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
