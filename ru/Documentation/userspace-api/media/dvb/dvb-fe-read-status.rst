************************************************
Запрос статуса и статистики фронтенда (frontend)
************************************************

После вызова :ref:`FE_SET_PROPERTY <FE_GET_PROPERTY>` фронтенд (frontend)
запустит поток ядра, который будет периодически проверять статус
захвата (lock) тюнера и предоставлять статистику о качестве
сигнала.

Информацию о статусе захвата тюнера фронтенда можно запросить
с помощью :ref:`FE_READ_STATUS`.

Статистика сигнала предоставляется через
:ref:`FE_GET_PROPERTY`.

.. note::

   Для большинства статистических показателей требуется, чтобы демодулятор
   был полностью захвачен (lock) (например, с установленным битом
   :c:type:`FE_HAS_LOCK <fe_status>`). Подробнее см.
   :ref:`Показатели статистики фронтенда <frontend-stat-properties>`.
