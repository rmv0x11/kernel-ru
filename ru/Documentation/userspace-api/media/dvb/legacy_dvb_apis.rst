.. _legacy_dvb_apis:

*****************************************
Устаревшие API цифрового телевидения (TV)
*****************************************

API, описанные здесь, **не следует** использовать в новых драйверах или приложениях.

Frontend API DVBv3 имеет проблемы с новыми системами доставки, включая
DVB-S2, DVB-T2, ISDB и т. д.

.. attention::

   API, описанные здесь, не обязательно отражают текущую
   реализацию кода, так как этот раздел документа был написан
   для DVB версии 1, тогда как код отражает реализацию
   DVB версии 3.
