========
DMX_STOP
========

Название
--------

DMX_STOP

Описание интерфейса
-------------------

.. c:macro:: DMX_STOP

``int ioctl(int fd, DMX_STOP)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

Описание
--------

Этот вызов ioctl используется для остановки текущей операции фильтрации,
заданной через вызовы ioctl :ref:`DMX_SET_FILTER` или :ref:`DMX_SET_PES_FILTER`
и запущенной командой :ref:`DMX_START`.

Возвращаемое значение
---------------------

При успешном выполнении возвращается 0.

При ошибке возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
