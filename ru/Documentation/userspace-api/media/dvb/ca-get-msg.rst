==========
CA_GET_MSG
==========

Имя
---

CA_GET_MSG

Синопсис
--------

.. c:macro:: CA_GET_MSG

``int ioctl(fd, CA_GET_MSG, struct ca_msg *msg)``

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

``msg``
  Указатель на структуру :c:type:`ca_msg`.

Описание
--------

Принимает сообщение через модуль CI CA.

.. note::

   Обратите внимание, что в большинстве драйверов это выполняется путём
   чтения из узла устройства /dev/adapter?/ca?.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
