===========
CA_SEND_MSG
===========

Имя
---

CA_SEND_MSG

Обзор
-----

.. c:macro:: CA_SEND_MSG

``int ioctl(fd, CA_SEND_MSG, struct ca_msg *msg)``

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

``msg``
  Указатель на структуру :c:type:`ca_msg`.

Описание
--------

Отправляет сообщение через модуль CI CA.

.. note::

   Обратите внимание, что в большинстве драйверов это делается путём записи
   в узел устройства /dev/adapter?/ca?.

Возвращаемое значение
---------------------

При успешном выполнении возвращается 0.

При ошибке возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
