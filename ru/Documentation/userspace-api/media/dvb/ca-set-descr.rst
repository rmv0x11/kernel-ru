.. _CA_SET_DESCR:

============
CA_SET_DESCR
============

Имя
---

CA_SET_DESCR

Синопсис
--------

.. c:macro:: CA_SET_DESCR

``int ioctl(fd, CA_SET_DESCR, struct ca_descr *desc)``

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

``msg``
  Указатель на структуру :c:type:`ca_descr`.

Описание
--------

CA_SET_DESCR используется для передачи дескремблерным CA-слотам ключей
дескремблирования (называемых контрольными словами).

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в разделе
:ref:`Общие коды ошибок <gen-errors>`.
