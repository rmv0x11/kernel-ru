**************************
FE_READ_UNCORRECTED_BLOCKS
**************************

Имя
===

FE_READ_UNCORRECTED_BLOCKS

.. attention:: Этот ioctl устарел.

Синопсис
========

.. c:macro:: FE_READ_UNCORRECTED_BLOCKS

``int ioctl(int fd, FE_READ_UNCORRECTED_BLOCKS, uint32_t *ublocks)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``ublocks``
    Общее число неисправленных блоков, обнаруженных драйвером к данному моменту.

Описание
========

Этот вызов ioctl возвращает число неисправленных блоков, обнаруженных драйвером
устройства за время его работы. Для получения осмысленных измерений следует
вычислять приращение счётчика блоков за конкретный интервал времени. Для этой
команды достаточно доступа к устройству только на чтение.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
