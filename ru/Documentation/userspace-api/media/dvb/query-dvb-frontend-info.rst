******************************
Запрос информации о фронтенде
******************************

Обычно первое, что нужно сделать после открытия фронтенда, — проверить
его возможности. Это делается с помощью
:ref:`FE_GET_INFO`. Данный ioctl перечислит
версию Digital TV API и другие характеристики фронтенда; его можно
открывать как в режиме только для чтения, так и в режиме чтения/записи.
