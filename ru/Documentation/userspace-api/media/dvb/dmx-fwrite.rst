===================================
write() демультиплексора Digital TV
===================================

Имя
---

write() демультиплексора Digital TV

Описание вызова
---------------

.. c:function:: ssize_t write(int fd, const void *buf, size_t count)

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

``buf``
     Буфер с данными для записи

``count``
    Количество байт в буфере

Описание
--------

Этот системный вызов предоставляется только логическим устройством
``/dev/dvb/adapter?/dvr?``, связанным с физическим устройством демультиплексора,
которое обеспечивает фактическую функциональность DVR. Он используется для
воспроизведения цифрового записанного транспортного потока (Transport Stream).
Соответствующие фильтры должны быть определены в связанном физическом устройстве
демультиплексора, ``/dev/dvb/adapter?/demux?``.
Объём передаваемых данных подразумевается значением count.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменная ``errno`` устанавливается
соответствующим образом.

.. tabularcolumns:: |p{2.5cm}|p{15.0cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0
    :widths: 1 16

    -  -  ``EWOULDBLOCK``
       -  Данные не были записаны. Это может произойти, если был указан
	  ``O_NONBLOCK`` и больше нет доступного места в буфере (если
	  ``O_NONBLOCK`` не указан, функция будет блокироваться до тех пор, пока
	  место в буфере не станет доступным).

    -  -  ``EBUSY``
       -  Этот код ошибки указывает на конфликтующие запросы.
	  Соответствующее устройство демультиплексора настроено на приём данных от
	  фронтенда (front-end). Убедитесь, что эти фильтры остановлены и что
	  фильтры со входом, установленным в ``DMX_IN_DVR``, запущены.

Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
