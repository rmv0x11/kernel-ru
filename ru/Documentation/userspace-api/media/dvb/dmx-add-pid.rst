===========
DMX_ADD_PID
===========

Имя
---

DMX_ADD_PID

Краткое описание
----------------

.. c:macro:: DMX_ADD_PID

``int ioctl(fd, DMX_ADD_PID, __u16 *pid)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``pid``
   Номер PID, по которому выполняется фильтрация.

Описание
--------

Этот вызов ioctl позволяет добавить несколько PID к фильтру транспортного потока,
ранее настроенному с помощью :ref:`DMX_SET_PES_FILTER`, при выводе, равном
:c:type:`DMX_OUT_TSDEMUX_TAP <dmx_output>`.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
