============================================
Закрытие демультиплексора Digital TV close()
============================================

Имя
---

Закрытие демультиплексора Digital TV close()

Синопсис
--------

.. c:function:: int close(int fd)

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом
  :c:func:`open()`.

Описание
--------

Этот системный вызов деактивирует и освобождает фильтр, который был
ранее выделен с помощью вызова :c:func:`open()`.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
