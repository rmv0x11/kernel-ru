***********
FE_READ_SNR
***********

Имя
===

FE_READ_SNR

.. attention:: Этот ioctl устарел.

Обзор
=====

.. c:macro:: FE_READ_SNR

``int ioctl(int fd, FE_READ_SNR, int16_t *snr)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``snr``
    Отношение сигнал/шум сохраняется в \*snr.

Описание
========

Этот вызов ioctl возвращает отношение сигнал/шум для сигнала, принимаемого
в данный момент фронтендом. Для этой команды достаточно доступа к устройству
только на чтение.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.
