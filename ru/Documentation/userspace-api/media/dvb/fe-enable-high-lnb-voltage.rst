********************************
ioctl FE_ENABLE_HIGH_LNB_VOLTAGE
********************************

Имя
===

FE_ENABLE_HIGH_LNB_VOLTAGE - выбор выходного уровня постоянного тока между обычными напряжениями LNBf и повышенными напряжениями LNBf.

Объявление
==========

.. c:macro:: FE_ENABLE_HIGH_LNB_VOLTAGE

``int ioctl(int fd, FE_ENABLE_HIGH_LNB_VOLTAGE, unsigned int high)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``high``
    Допустимые флаги:

    -  0 - обычные 13 В и 18 В.

    -  >0 - включает немного повышенные напряжения вместо 13/18 В, чтобы
       компенсировать длинные антенные кабели.

Описание
========

Выбирает выходной уровень постоянного тока между обычными напряжениями LNBf
и повышенными напряжениями LNBf: значение 0 (обычное) либо значение больше 0
для повышенных напряжений.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
