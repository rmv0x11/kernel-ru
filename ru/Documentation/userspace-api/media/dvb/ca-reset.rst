========
CA_RESET
========

Имя
---

CA_RESET

Синопсис
--------

.. c:macro:: CA_RESET

``int ioctl(fd, CA_RESET)``

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

Описание
--------

Приводит оборудование условного доступа (Conditional Access) в его начальное
состояние. Должно вызываться перед началом использования оборудования CA.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменная ``errno`` устанавливается
соответствующим образом.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
