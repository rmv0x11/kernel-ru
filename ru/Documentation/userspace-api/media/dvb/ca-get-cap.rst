==========
CA_GET_CAP
==========

Имя
---

CA_GET_CAP

Обзор
-----

.. c:macro:: CA_GET_CAP

``int ioctl(fd, CA_GET_CAP, struct ca_caps *caps)``

Аргументы
---------

``fd``
  Файловый дескриптор, возвращённый предыдущим вызовом :c:func:`open()`.

``caps``
  Указатель на struct :c:type:`ca_caps`.

Описание
--------

Запрашивает у ядра информацию о доступных слотах CA и дескремблера, а также
об их типах.

Возвращаемое значение
---------------------

При успешном выполнении возвращается 0, а структура :c:type:`ca_caps`
заполняется.

При ошибке возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
