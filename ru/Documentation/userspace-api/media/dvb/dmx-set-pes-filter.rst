==================
DMX_SET_PES_FILTER
==================

Название
--------

DMX_SET_PES_FILTER

Синопсис
--------

.. c:macro:: DMX_SET_PES_FILTER

``int ioctl(int fd, DMX_SET_PES_FILTER, struct dmx_pes_filter_params *params)``

Аргументы
---------

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``params``
    Указатель на структуру, содержащую параметры фильтра.

Описание
--------

Этот вызов ioctl настраивает PES-фильтр в соответствии с предоставленными
параметрами. Под PES-фильтром понимается фильтр, основанный только на
идентификаторе пакета (PID), то есть фильтрация по заголовку PES или по
полезной нагрузке не поддерживается.

Возвращаемое значение
---------------------

В случае успеха возвращается 0.

В случае ошибки возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.
