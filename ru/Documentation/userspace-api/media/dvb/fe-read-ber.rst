***********
FE_READ_BER
***********

Имя
===

FE_READ_BER

.. attention:: Этот ioctl устарел.

Синопсис
========

.. c:macro:: FE_READ_BER

``int ioctl(int fd, FE_READ_BER, uint32_t *ber)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``ber``
    Частота битовых ошибок сохраняется в \*ber.

Описание
========

Этот вызов ioctl возвращает частоту битовых ошибок для сигнала, который в
данный момент принимается/демодулируется фронтендом. Для этой команды
достаточно доступа к устройству только для чтения.

Возвращаемое значение
=====================

При успешном выполнении возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
