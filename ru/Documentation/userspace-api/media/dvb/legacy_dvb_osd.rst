==========================
Устройство DVB OSD
==========================

.. attention:: **Не** используйте в новых драйверах!
             См.: :ref:`legacy_dvb_decoder_notes`

Устройство DVB OSD управляет экранным меню (OnScreen-Display) DVB-карт на
базе AV7110 с аппаратным декодером MPEG2. Доступ к нему осуществляется через
``/dev/dvb/adapter?/osd0``.
Доступ к типам данных и определениям ioctl можно получить, включив
``linux/dvb/osd.h`` в ваше приложение.

OSD не является фрейм-буфером, как на многих других картах.
Это своего рода холст, на котором можно рисовать.
Глубина цвета ограничена в зависимости от объёма установленной памяти.
Необходимо настроить соответствующую палитру цветов.
Объём установленной памяти можно определить с помощью ioctl
`OSD_GET_CAPABILITY`_.

Типы данных OSD
===============

OSD_Command
-----------

Краткое описание
~~~~~~~~~~~~~~~~~

.. code-block:: c

    typedef enum {
	/* All functions return -2 on "not open" */
	OSD_Close = 1,
	OSD_Open,
	OSD_Show,
	OSD_Hide,
	OSD_Clear,
	OSD_Fill,
	OSD_SetColor,
	OSD_SetPalette,
	OSD_SetTrans,
	OSD_SetPixel,
	OSD_GetPixel,
	OSD_SetRow,
	OSD_SetBlock,
	OSD_FillRow,
	OSD_FillBlock,
	OSD_Line,
	OSD_Query,
	OSD_Test,
	OSD_Text,
	OSD_SetWindow,
	OSD_MoveWindow,
	OSD_OpenRaw,
    } OSD_Command;

Команды
~~~~~~~

.. note::  Все функции возвращают -2 при состоянии "не открыто"

.. flat-table::
    :header-rows:  1
    :stub-columns: 0

    -  ..

       -  Команда

       -  | Используемые переменные ``struct`` `osd_cmd_t`_.
          | Usage{variable} при альтернативном использовании.

       -  :cspan:`2` Описание



    -  ..

       -  ``OSD_Close``

       -  -

       -  | Отключает OSD и освобождает буферы.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_Open``

       -  | x0,y0,x1,y1,
          | BitPerPixel[2/4/8]{color&0x0F},
          | mix[0..15]{color&0xF0}

       -  | Открывает OSD с указанным размером и глубиной цвета
          | Возвращает 0 при успехе,
          | -1 при ошибке выделения DRAM,
          | -2 при состоянии "уже открыто".

    -  ..

       -  ``OSD_Show``

       - -

       -  | Включает режим OSD.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_Hide``

       - -

       -  | Отключает режим OSD.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_Clear``

       - -

       -  | Устанавливает все пиксели в цвет 0.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_Fill``

       -  color

       -  | Устанавливает все пиксели в цвет <color>.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_SetColor``

       -  | color,
          | R{x0},G{y0},B{x1},
          | opacity{y1}

       -  | Устанавливает запись палитры <num> в <r,g,b>, применяются <mix> и <trans>
          | R,G,B: 0..255
          | R=Red, G=Green, B=Blue
          | opacity=0:      непрозрачность пикселя 0% (показывается только видеопиксель)
          | opacity=1..254: непрозрачность пикселя как указано в заголовке
          | opacity=255:    непрозрачность пикселя 100% (показывается только пиксель OSD)
          | Возвращает 0 при успехе, -1 при ошибке.

    -  ..

       -  ``OSD_SetPalette``

       -  | firstcolor{color},
          | lastcolor{x0},data

       -  | Устанавливает несколько записей в палитре.
          | Устанавливает записи с "firstcolor" по "lastcolor" из
            массива "data".
          | Data содержит 4 байта для каждого цвета:
          | R,G,B и значение непрозрачности: 0->прозрачный, 1..254->смешивание,
            255->пиксель

    -  ..

       -  ``OSD_SetTrans``

       -  transparency{color}

       -  | Устанавливает прозрачность смешанного пикселя (0..15).
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_SetPixel``

       -  x0,y0,color

       -  | Устанавливает пиксель <x>,<y> в цвет с номером <color>.
          | Возвращает 0 при успехе, -1 при ошибке.

    -  ..

       -  ``OSD_GetPixel``

       -  x0,y0

       -  | Возвращает номер цвета пикселя <x>,<y> или -1.
          | Команда в настоящее время не поддерживается AV7110!

    -  ..

       -  ``OSD_SetRow``

       -  x0,y0,x1,data

       -  | Заполняет пиксели от x0,y до x1,y содержимым data[].
          | Возвращает 0 при успехе, -1 при отсечении всех пикселей (ни один пиксель
            не нарисован).

    -  ..

       -  ``OSD_SetBlock``

       -  | x0,y0,x1,y1,
          | increment{color},
          | data

       -  | Заполняет пиксели от x0,y0 до x1,y1 содержимым data[].
          | Inc содержит ширину одной строки в блоке данных,
          | inc<=0 использует ширину блока как ширину строки.
          | Возвращает 0 при успехе, -1 при отсечении всех пикселей.

    -  ..

       -  ``OSD_FillRow``

       -  x0,y0,x1,color

       -  | Заполняет пиксели от x0,y до x1,y цветом <color>.
          | Возвращает 0 при успехе, -1 при отсечении всех пикселей.

    -  ..

       -  ``OSD_FillBlock``

       -  x0,y0,x1,y1,color

       -  | Заполняет пиксели от x0,y0 до x1,y1 цветом <color>.
          | Возвращает 0 при успехе, -1 при отсечении всех пикселей.

    -  ..

       -  ``OSD_Line``

       -  x0,y0,x1,y1,color

       -  | Рисует линию от x0,y0 до x1,y1 цветом <color>.
          | Возвращает 0 при успехе.

    -  ..

       -  ``OSD_Query``

       -  | x0,y0,x1,y1,
          | xasp{color}; yasp=11

       -  | Заполняет параметры размерами изображения и соотношением сторон
            пикселя.
          | Возвращает 0 при успехе.
          | Команда в настоящее время не поддерживается AV7110!

    -  ..

       -  ``OSD_Test``

       -  -

       -  | Рисует тестовое изображение.
          | Только для целей отладки.
          | Возвращает 0 при успехе.
    -  ..

       -  ``OSD_Text``

       -  x0,y0,size,color,text

       -  Рисует текст в позиции x0,y0 цветом <color>.

    -  ..

       -  ``OSD_SetWindow``

       -  x0

       -  Устанавливает окно с номером 0<x0<8 как текущее.

    -  ..

       -  ``OSD_MoveWindow``

       -  x0,y0

       -  Перемещает текущее окно в (x0, y0).

    -  ..

       -  ``OSD_OpenRaw``

       -  | x0,y0,x1,y1,
          | `osd_raw_window_t`_ {color}

       -  Открывает другие типы окон OSD.

Описание
~~~~~~~~

Тип данных ``OSD_Command`` используется с ioctl `OSD_SEND_CMD`_, чтобы
сообщить драйверу, какую команду OSD_Command выполнить.

osd_cmd_t
---------

Краткое описание
~~~~~~~~~~~~~~~~~

.. code-block:: c

    typedef struct osd_cmd_s {
	OSD_Command cmd;
	int x0;
	int y0;
	int x1;
	int y1;
	int color;
	void __user *data;
    } osd_cmd_t;

Переменные
~~~~~~~~~~

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    -  ..

       -  ``OSD_Command cmd``

       -  `OSD_Command`_ для выполнения.

    -  ..

       -  ``int x0``

       -  Первая горизонтальная позиция.

    -  ..

       -  Первая вертикальная позиция.

    -  ..

    -  ..

       -  Номер цвета в палитре.

    -  ..

       -  ``void __user *data``

Описание
~~~~~~~~

Тип данных ``osd_cmd_t`` используется с ioctl `OSD_SEND_CMD`_.
Он содержит данные для команды OSD_Command и саму `OSD_Command`_.
Структура должна быть передана драйверу, и её компоненты могут быть
изменены им.

~~~~~~~~~

.. flat-table::
    :header-rows:  0
