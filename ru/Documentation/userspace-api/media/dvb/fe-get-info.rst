*****************
ioctl FE_GET_INFO
*****************

Имя
===

FE_GET_INFO - Запрашивает возможности фронтенда цифрового ТВ и возвращает
сведения о фронтенде. Этому вызову требуется только доступ к устройству на
чтение.

Синопсис
========

.. c:macro:: FE_GET_INFO

``int ioctl(int fd, FE_GET_INFO, struct dvb_frontend_info *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``argp``
    указатель на структуру :c:type:`dvb_frontend_info`

Описание
========

Все фронтенд-устройства цифрового ТВ поддерживают ioctl :ref:`FE_GET_INFO`. Он
используется для идентификации устройств ядра, совместимых с данной
спецификацией, и для получения сведений о возможностях драйвера и оборудования.
Этот ioctl принимает указатель на dvb_frontend_info, который заполняется
драйвером. Если драйвер несовместим с данной спецификацией, ioctl возвращает
ошибку.

возможности фронтенда
=====================

Возможности описывают то, что может делать фронтенд. Некоторые возможности
поддерживаются только на отдельных конкретных типах фронтендов.

Возможности фронтенда описаны в :c:type:`fe_caps`.

Возвращаемое значение
=====================

При успехе возвращается 0.

При ошибке возвращается -1, и соответствующим образом устанавливается переменная
``errno``.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
