*****************
ioctl FE_SET_TONE
*****************

Имя
===

FE_SET_TONE - Включает/выключает генерацию непрерывного тона 22 кГц.

Синопсис
========

.. c:macro:: FE_SET_TONE

``int ioctl(int fd, FE_SET_TONE, enum fe_sec_tone_mode tone)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый :c:func:`open()`.

``tone``
    Целочисленное перечисляемое значение, описанное в :c:type:`fe_sec_tone_mode`

Описание
========

Этот ioctl используется для управления генерацией непрерывного тона 22 кГц.
Данный вызов требует прав на чтение/запись.

Обычно подсистемы спутниковых антенн требуют, чтобы устройство цифрового ТВ
посылало тон 22 кГц для выбора между верхним/нижним диапазоном на некоторых
двухдиапазонных LNBf. Он также используется для отправки сигналов оборудованию
DiSEqC, но это делается с помощью ioctl-ов DiSEqC.

.. attention:: Если к одной и той же антенне подключено более одного устройства,
   установка тона может создавать помехи для других устройств, поскольку они могут
   потерять возможность выбора диапазона. Поэтому рекомендуется, чтобы приложения
   переключались в SEC_TONE_OFF, когда устройство не используется.

Возвращаемое значение
=====================

При успехе возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.

Общие коды ошибок описаны в главе
:ref:`Общие коды ошибок <gen-errors>`.
