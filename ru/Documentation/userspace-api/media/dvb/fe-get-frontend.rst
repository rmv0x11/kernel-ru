***************
FE_GET_FRONTEND
***************

Имя
===

FE_GET_FRONTEND

.. attention:: Этот ioctl является устаревшим.

Обзор
=====

.. c:macro:: FE_GET_FRONTEND

``int ioctl(int fd, FE_GET_FRONTEND, struct dvb_frontend_parameters *p)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращаемый функцией :c:func:`open()`.

``p``
    Указывает на параметры операции настройки.

Описание
========

Этот вызов ioctl запрашивает действующие в данный момент параметры
фронтенда. Для этой команды достаточно доступа к устройству только для чтения.

Возвращаемое значение
=====================

При успешном выполнении возвращается 0.

При ошибке возвращается -1, и переменной ``errno`` присваивается
соответствующее значение.
