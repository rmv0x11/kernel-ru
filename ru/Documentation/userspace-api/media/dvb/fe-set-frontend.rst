***************
FE_SET_FRONTEND
***************

.. attention:: Этот ioctl является устаревшим.

Имя
===

FE_SET_FRONTEND

Обзор
=====

.. c:macro:: FE_SET_FRONTEND

``int ioctl(int fd, FE_SET_FRONTEND, struct dvb_frontend_parameters *p)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``p``
    Указывает на параметры операции настройки (tuning).

Описание
========

Этот вызов ioctl запускает операцию настройки (tuning) с использованием
указанных параметров. Результат данного вызова будет успешным, если
параметры были корректны и настройку удалось инициировать. Однако результат
самой операции настройки поступит асинхронно в виде события (см.
документацию по :ref:`FE_GET_EVENT` и
FrontendEvent.) Если новая операция :ref:`FE_SET_FRONTEND`
инициируется до того, как завершилась предыдущая, предыдущая операция будет
прервана в пользу новой. Эта команда требует доступа к устройству на
чтение/запись.

Возвращаемое значение
=====================

В случае успеха возвращается 0.

В случае ошибки возвращается -1, а переменной ``errno`` присваивается
соответствующее значение.
