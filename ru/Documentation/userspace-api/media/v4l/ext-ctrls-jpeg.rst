.. _jpeg-controls:

****************************************
Справочник по элементам управления JPEG
****************************************

Класс JPEG включает элементы управления общими возможностями кодировщиков
и декодировщиков JPEG. В настоящее время он включает возможности для кодеков,
реализующих процесс прогрессивного базового сжатия DCT с энтропийным
кодированием по Хаффману.


.. _jpeg-control-id:

Идентификаторы элементов управления JPEG
========================================

``V4L2_CID_JPEG_CLASS (class)``
    Дескриптор класса JPEG. Вызов
    :ref:`VIDIOC_QUERYCTRL` для этого элемента управления
    вернёт описание данного класса элементов управления.

``V4L2_CID_JPEG_CHROMA_SUBSAMPLING (menu)``
    Коэффициенты субдискретизации цветности описывают, как каждая компонента
    входного изображения дискретизируется относительно максимальной частоты
    дискретизации в каждом пространственном измерении. Подробнее см. :ref:`itu-t81`,
    пункт A.1.1. Элемент управления ``V4L2_CID_JPEG_CHROMA_SUBSAMPLING`` определяет,
    как компоненты Cb и Cr подвергаются субдискретизации после преобразования
    входного изображения из цветового пространства RGB в Y'CbCr.

.. tabularcolumns:: |p{7.5cm}|p{10.0cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_444``
      - Без субдискретизации цветности, каждый пиксель имеет значения Y, Cr и Cb.
    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_422``
      - Субдискретизация компонент Cr, Cb по горизонтали с коэффициентом 2.
    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_420``
      - Субдискретизация компонент Cr, Cb по горизонтали и вертикали с коэффициентом 2.
    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_411``
      - Субдискретизация компонент Cr, Cb по горизонтали с коэффициентом 4.
    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_410``
      - Субдискретизация компонент Cr, Cb по горизонтали с коэффициентом 4 и по вертикали с коэффициентом 2.
    * - ``V4L2_JPEG_CHROMA_SUBSAMPLING_GRAY``
      - Использовать только компоненту яркости.



``V4L2_CID_JPEG_RESTART_INTERVAL (integer)``
    Интервал перезапуска определяет интервал вставки маркеров RSTm
    (m = 0..7). Назначение этих маркеров — дополнительно
    реинициализировать процесс кодирования, чтобы обрабатывать блоки
    изображения независимо. Для процессов сжатия с потерями единицей
    интервала перезапуска является MCU (Minimum Coded Unit), и его значение
    содержится в маркере DRI (Define Restart Interval). Если элемент управления
    ``V4L2_CID_JPEG_RESTART_INTERVAL`` установлен в 0, маркеры DRI и RSTm
    вставляться не будут.

.. _jpeg-quality-control:

``V4L2_CID_JPEG_COMPRESSION_QUALITY (integer)``
    Определяет компромисс между качеством и размером изображения.
    Он предоставляет приложениям более простой способ управления качеством
    изображения без необходимости прямой переконфигурации таблиц квантования
    яркости и цветности. В случаях, когда драйвер использует таблицы квантования,
    настроенные непосредственно приложением с помощью интерфейсов, определённых
    в другом месте, элемент управления ``V4L2_CID_JPEG_COMPRESSION_QUALITY`` должен быть
    установлен драйвером в 0.

    Диапазон значений этого элемента управления зависит от драйвера. Имеют смысл только
    положительные, ненулевые значения. Рекомендуемый диапазон — от 1 до 100,
    где большие значения соответствуют лучшему качеству изображения.

.. _jpeg-active-marker-control:

``V4L2_CID_JPEG_ACTIVE_MARKER (bitmask)``
    Указывает, какие маркеры JPEG включаются в сжатый поток. Этот
    элемент управления действителен только для кодировщиков.



.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_JPEG_ACTIVE_MARKER_APP0``
      - Сегмент данных приложения APP\ :sub:`0`.
    * - ``V4L2_JPEG_ACTIVE_MARKER_APP1``
      - Сегмент данных приложения APP\ :sub:`1`.
    * - ``V4L2_JPEG_ACTIVE_MARKER_COM``
      - Сегмент комментариев.
    * - ``V4L2_JPEG_ACTIVE_MARKER_DQT``
      - Сегмент таблиц квантования.
    * - ``V4L2_JPEG_ACTIVE_MARKER_DHT``
      - Сегмент таблиц Хаффмана.



Более подробные сведения о спецификации JPEG см. в :ref:`itu-t81`,
:ref:`jfif`, :ref:`w3c-jpeg-jfif`.
