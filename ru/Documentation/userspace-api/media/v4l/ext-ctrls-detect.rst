.. _detect-controls:

***********************************
Справочник по управлению Detect
***********************************

Класс Detect включает элементы управления для распространённых возможностей
различных устройств, способных обнаруживать движение или объекты.


.. _detect-control-id:

Идентификаторы элементов управления Detect
==========================================

``V4L2_CID_DETECT_CLASS (class)``
    Дескриптор класса Detect. Вызов
    :ref:`VIDIOC_QUERYCTRL` для этого элемента управления
    вернёт описание данного класса элементов управления.

``V4L2_CID_DETECT_MD_MODE (menu)``
    Задаёт режим обнаружения движения.

.. tabularcolumns:: |p{7.7cm}|p{9.8cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_DETECT_MD_MODE_DISABLED``
      - Отключить обнаружение движения.
    * - ``V4L2_DETECT_MD_MODE_GLOBAL``
      - Использовать единый порог обнаружения движения.
    * - ``V4L2_DETECT_MD_MODE_THRESHOLD_GRID``
      - Изображение делится на сетку, каждая ячейка которой имеет собственный
	порог обнаружения движения. Эти пороги задаются через матричный элемент
	управления ``V4L2_CID_DETECT_MD_THRESHOLD_GRID``.
    * - ``V4L2_DETECT_MD_MODE_REGION_GRID``
      - Изображение делится на сетку, каждая ячейка которой имеет собственное
	значение региона, определяющее, какие пороги обнаружения движения для
	данного региона должны использоваться. Каждый регион имеет свои
	собственные пороги. Способ настройки этих порогов для регионов зависит
	от драйвера. Значения регионов для сетки задаются через матричный элемент
	управления ``V4L2_CID_DETECT_MD_REGION_GRID``.



``V4L2_CID_DETECT_MD_GLOBAL_THRESHOLD (integer)``
    Задаёт глобальный порог обнаружения движения, используемый в режиме
    обнаружения движения ``V4L2_DETECT_MD_MODE_GLOBAL``.

``V4L2_CID_DETECT_MD_THRESHOLD_GRID (__u16 matrix)``
    Задаёт пороги обнаружения движения для каждой ячейки сетки. Используется
    с режимом обнаружения движения ``V4L2_DETECT_MD_MODE_THRESHOLD_GRID``.
    Элемент матрицы (0, 0) соответствует ячейке в левом верхнем углу сетки.

``V4L2_CID_DETECT_MD_REGION_GRID (__u8 matrix)``
    Задаёт значение региона обнаружения движения для каждой ячейки сетки.
    Используется с режимом обнаружения движения
    ``V4L2_DETECT_MD_MODE_REGION_GRID``. Элемент матрицы (0, 0) соответствует
    ячейке в левом верхнем углу сетки.
