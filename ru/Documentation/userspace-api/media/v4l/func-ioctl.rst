************
V4L2 ioctl()
************

Имя
===

v4l2-ioctl - Программирование устройства V4L2

Синопсис
========

.. code-block:: c

    #include <sys/ioctl.h>

``int ioctl(int fd, int request, void *argp)``

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

``request``
    Код ioctl-запроса V4L2, как определено в заголовочном файле
    ``videodev2.h``, например VIDIOC_QUERYCAP.

``argp``
    Указатель на параметр функции, обычно на структуру.

Описание
========

Функция :ref:`ioctl() <func-ioctl>` используется для программирования устройств
V4L2. Аргумент ``fd`` должен быть открытым файловым дескриптором. В ioctl-запросе
``request`` закодировано, является ли аргумент входным, выходным параметром или
параметром чтения/записи, а также размер аргумента ``argp`` в байтах. Макросы и
определения, задающие ioctl-запросы V4L2, расположены в заголовочном файле
``videodev2.h``. Приложениям следует использовать свою собственную копию, а не
подключать версию из исходных текстов ядра в системе, в которой они компилируются.
Все ioctl-запросы V4L2, их соответствующие функции и параметры описаны в
:ref:`user-func`.

Возвращаемое значение
=====================

В случае успеха возвращается 0, в случае ошибки — -1, и переменной ``errno``
присваивается соответствующее значение. Универсальные коды ошибок описаны в главе
:ref:`Generic Error Codes <gen-errors>`.

Когда ioctl, принимающий выходной параметр или параметр чтения/записи, завершается
с ошибкой, этот параметр остаётся без изменений.
