.. _func-close:

************
V4L2 close()
************

Имя
===

v4l2-close - Закрыть устройство V4L2

Синопсис
========

.. code-block:: c

    #include <unistd.h>

.. c:function:: int close( int fd )

Аргументы
=========

``fd``
    Файловый дескриптор, возвращённый :c:func:`open()`.

Описание
========

Закрывает устройство. Любой выполняющийся ввод-вывод прерывается, а ресурсы,
связанные с файловым дескриптором, освобождаются. Однако параметры формата
данных, текущий вход или выход, значения элементов управления и другие
свойства остаются неизменными.

Возвращаемое значение
=====================

Функция возвращает 0 в случае успеха, -1 при неудаче, при этом ``errno``
устанавливается соответствующим образом. Возможные коды ошибок:

EBADF
    ``fd`` не является корректным открытым файловым дескриптором.
