************************************************
Справочник по элементам управления колориметрией
************************************************

Класс колориметрии (Colorimetry) включает элементы управления для формирования
изображений с расширенным динамическим диапазоном (High Dynamic Range),
предназначенные для представления цветов в цифровых изображениях и видео. Эти
элементы управления следует использовать при кодировании и декодировании видео
и изображений, а также в приёмниках и передатчиках HDMI.

Идентификаторы элементов управления колориметрией
-------------------------------------------------

.. _colorimetry-control-id:

``V4L2_CID_COLORIMETRY_CLASS (class)``
    Дескриптор класса колориметрии. Вызов
    :ref:`VIDIOC_QUERYCTRL` для этого элемента управления
    вернёт описание данного класса элементов управления.

``V4L2_CID_COLORIMETRY_HDR10_CLL_INFO (struct)``
    Content Light Level задаёт верхние границы номинального целевого уровня
    яркости изображений.

.. c:type:: v4l2_ctrl_hdr10_cll_info

.. cssclass:: longtable

.. flat-table:: struct v4l2_ctrl_hdr10_cll_info
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 1 2

    * - __u16
      - ``max_content_light_level``
      - Верхняя граница максимального уровня яркости среди всех отдельных
        отсчётов для изображений видеопоследовательности, cd/m\ :sup:`2`.
        При значении, равном 0, такая верхняя граница отсутствует.
    * - __u16
      - ``max_pic_average_light_level``
      - Верхняя граница максимального среднего уровня яркости среди
        отсчётов для любого отдельного изображения видеопоследовательности,
        cd/m\ :sup:`2`. При значении, равном 0, такая верхняя граница отсутствует.

``V4L2_CID_COLORIMETRY_HDR10_MASTERING_DISPLAY (struct)``
    Mastering display задаёт цветовой объём (цветовые координаты основных цветов,
    точку белого и диапазон яркости) дисплея, который считается эталонным
    (mastering) дисплеем для текущего видеоконтента.

.. c:type:: v4l2_ctrl_hdr10_mastering_display

.. cssclass:: longtable

.. flat-table:: struct v4l2_ctrl_hdr10_mastering_display
    :header-rows:  0
    :stub-columns: 0
    :widths:       1 1 2

    * - __u16
      - ``display_primaries_x[3]``
      - Задаёт нормированную координату цветности x основного цветового
        компонента c эталонного дисплея с шагом 0.00002.
        При описании эталонного дисплея, использующего основные цвета Red, Green
        и Blue, значение индекса c, равное 0, соответствует основному цвету Green,
        c, равное 1, соответствует основному цвету Blue, а c, равное 2,
        соответствует основному цвету Red.
    * - __u16
      - ``display_primaries_y[3]``
      - Задаёт нормированную координату цветности y основного цветового
        компонента c эталонного дисплея с шагом 0.00002.
        При описании эталонного дисплея, использующего основные цвета Red, Green
        и Blue, значение индекса c, равное 0, соответствует основному цвету Green,
        c, равное 1, соответствует основному цвету Blue, а c, равное 2,
        соответствует основному цвету Red.
    * - __u16
      - ``white_point_x``
      - Задаёт нормированную координату цветности x точки белого эталонного
        дисплея с шагом 0.00002.
    * - __u16
      - ``white_point_y``
      - Задаёт нормированную координату цветности y точки белого эталонного
        дисплея с шагом 0.00002.
    * - __u32
      - ``max_luminance``
      - Задаёт номинальную максимальную яркость эталонного дисплея в единицах
        0.0001 cd/m\ :sup:`2`.
    * - __u32
      - ``min_luminance``
      - Задаёт номинальную минимальную яркость эталонного дисплея в единицах
        0.0001 cd/m\ :sup:`2`.
