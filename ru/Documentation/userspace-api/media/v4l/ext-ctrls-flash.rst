.. _flash-controls:

*******************************************
Справочник по элементам управления вспышкой
*******************************************

Элементы управления вспышкой V4L2 предназначены для предоставления универсального
доступа к устройствам контроллеров вспышки. Устройства контроллеров вспышки обычно
используются в цифровых камерах.

Этот интерфейс может поддерживать как светодиодные (LED), так и ксеноновые устройства
вспышки. На момент написания этого текста драйвера ксеноновой вспышки, использующего
данный интерфейс, не существует.


.. _flash-controls-use-cases:

Поддерживаемые сценарии использования
=====================================


Несинхронизированная светодиодная вспышка (программный строб)
-------------------------------------------------------------

Несинхронизированная светодиодная вспышка управляется непосредственно хостом, как и
сенсор. Вспышка должна быть включена хостом до начала экспозиции изображения и
выключена по её окончании. За временные характеристики вспышки полностью отвечает хост.

Пример такого устройства: Nokia N900.


Синхронизированная светодиодная вспышка (аппаратный строб)
----------------------------------------------------------

Синхронизированная светодиодная вспышка предварительно программируется хостом (мощность
и тайм-аут), но управляется сенсором посредством строб-сигнала, передаваемого от сенсора
к вспышке.

Сенсор управляет длительностью и временными характеристиками вспышки. Эта информация
обычно должна быть доступна сенсору.


Светодиодная вспышка в режиме фонарика
--------------------------------------

Светодиодная вспышка может использоваться в качестве фонарика как совместно с другим
сценарием, связанным с камерой, так и по отдельности.


.. _flash-control-id:

Идентификаторы элементов управления вспышкой
--------------------------------------------

``V4L2_CID_FLASH_CLASS (class)``
    Дескриптор класса FLASH.

.. _v4l2-cid-flash-led-mode:

``V4L2_CID_FLASH_LED_MODE (menu)``
    Определяет режим работы светодиода вспышки — мощного белого светодиода,
    подключённого к контроллеру вспышки. Установка этого элемента управления может
    быть невозможна при наличии некоторых неисправностей. См. V4L2_CID_FLASH_FAULT.


.. tabularcolumns:: |p{5.7cm}|p{11.8cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_FLASH_LED_MODE_NONE``
      - Выключено.
    * - ``V4L2_FLASH_LED_MODE_FLASH``
      - Режим вспышки.
    * - ``V4L2_FLASH_LED_MODE_TORCH``
      - Режим фонарика.

        См. V4L2_CID_FLASH_TORCH_INTENSITY.



.. _v4l2-cid-flash-strobe-source:

``V4L2_CID_FLASH_STROBE_SOURCE (menu)``
    Определяет источник строба светодиода вспышки.

.. tabularcolumns:: |p{7.5cm}|p{7.5cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_FLASH_STROBE_SOURCE_SOFTWARE``
      - Строб вспышки запускается с помощью элемента управления
	V4L2_CID_FLASH_STROBE.
    * - ``V4L2_FLASH_STROBE_SOURCE_EXTERNAL``
      - Строб вспышки запускается внешним источником. Обычно это сенсор, что
	позволяет синхронизировать начало строба вспышки с началом экспозиции.
        Этот способ управления стробом светодиода вспышки имеет два дополнительных
        предварительных условия: :ref:`строб-выход <v4l2-cid-flash-strobe-oe>`
        источника строба должен быть включён (если он доступен),
        а :ref:`режим светодиода вспышки <v4l2-cid-flash-led-mode>`
        контроллера вспышки должен быть установлен в
        ``V4L2_FLASH_LED_MODE_FLASH``.



``V4L2_CID_FLASH_STROBE (button)``
    Запуск строба вспышки. Действителен, когда V4L2_CID_FLASH_LED_MODE установлен в
    V4L2_FLASH_LED_MODE_FLASH, а V4L2_CID_FLASH_STROBE_SOURCE
    установлен в V4L2_FLASH_STROBE_SOURCE_SOFTWARE. Установка этого
    элемента управления может быть невозможна при наличии некоторых неисправностей. См.
    V4L2_CID_FLASH_FAULT.

``V4L2_CID_FLASH_STROBE_STOP (button)``
    Немедленно остановить строб вспышки.

``V4L2_CID_FLASH_STROBE_STATUS (boolean)``
    Состояние строба: стробирует ли вспышка в данный момент или нет.
    Это элемент управления, доступный только для чтения.

``V4L2_CID_FLASH_TIMEOUT (integer)``
    Аппаратный тайм-аут для вспышки. Строб вспышки останавливается по истечении этого
    промежутка времени с момента начала строба.

``V4L2_CID_FLASH_INTENSITY (integer)``
    Интенсивность строба вспышки, когда светодиод вспышки находится в режиме вспышки
    (V4L2_FLASH_LED_MODE_FLASH). По возможности единицей измерения должны быть
    миллиамперы (мА).

``V4L2_CID_FLASH_TORCH_INTENSITY (integer)``
    Интенсивность светодиода вспышки в режиме фонарика
    (V4L2_FLASH_LED_MODE_TORCH). По возможности единицей измерения должны быть
    миллиамперы (мА). Установка этого элемента управления может быть невозможна при
    наличии некоторых неисправностей. См. V4L2_CID_FLASH_FAULT.

``V4L2_CID_FLASH_INDICATOR_INTENSITY (integer)``
    Интенсивность светодиода индикатора. Светодиод индикатора может быть полностью
    независим от светодиода вспышки. По возможности единицей измерения должны быть
    микроамперы (мкА).

``V4L2_CID_FLASH_FAULT (bitmask)``
    Неисправности, связанные со вспышкой. Неисправности сообщают о конкретных проблемах
    в самой микросхеме вспышки или в подключённых к ней светодиодах. Неисправности могут
    препятствовать дальнейшему использованию некоторых элементов управления вспышкой. В
    частности, V4L2_CID_FLASH_LED_MODE устанавливается в V4L2_FLASH_LED_MODE_NONE,
    если неисправность затрагивает светодиод вспышки. Какие именно неисправности дают
    такой эффект — зависит от микросхемы. Чтение неисправностей сбрасывает элемент
    управления и по возможности возвращает микросхему в работоспособное состояние.

.. tabularcolumns:: |p{8.4cm}|p{9.1cm}|

.. flat-table::
    :header-rows:  0
    :stub-columns: 0

    * - ``V4L2_FLASH_FAULT_OVER_VOLTAGE``
      - Напряжение, подаваемое контроллером вспышки на светодиод вспышки, превысило
	предел, специфичный для данного контроллера вспышки.
    * - ``V4L2_FLASH_FAULT_TIMEOUT``
      - Строб вспышки всё ещё был включён, когда истёк тайм-аут, заданный пользователем ---
	элемент управления V4L2_CID_FLASH_TIMEOUT. Не все контроллеры вспышки могут
	устанавливать это при всех таких условиях.
    * - ``V4L2_FLASH_FAULT_OVER_TEMPERATURE``
      - Контроллер вспышки перегрелся.
    * - ``V4L2_FLASH_FAULT_SHORT_CIRCUIT``
      - Сработала защита контроллера вспышки от короткого замыкания.
    * - ``V4L2_FLASH_FAULT_OVER_CURRENT``
      - Ток в цепи питания светодиода превысил предел, специфичный для данного
	контроллера вспышки.
    * - ``V4L2_FLASH_FAULT_INDICATOR``
      - Контроллер вспышки обнаружил состояние короткого замыкания или обрыва цепи
	на светодиоде индикатора.
    * - ``V4L2_FLASH_FAULT_UNDER_VOLTAGE``
      - Напряжение, подаваемое контроллером вспышки на светодиод вспышки, опустилось
	ниже минимального предела, специфичного для данного контроллера вспышки.
    * - ``V4L2_FLASH_FAULT_INPUT_VOLTAGE``
      - Входное напряжение контроллера вспышки ниже предела, при котором стробирование
	вспышки на полном токе будет невозможно. Это состояние сохраняется до тех пор,
	пока данный флаг не будет снят.
    * - ``V4L2_FLASH_FAULT_LED_OVER_TEMPERATURE``
      - Температура светодиода превысила допустимый верхний предел.



``V4L2_CID_FLASH_CHARGE (boolean)``
    Включить или отключить зарядку конденсатора ксеноновой вспышки.

``V4L2_CID_FLASH_READY (boolean)``
    Готова ли вспышка к стробированию? Ксеноновым вспышкам требуется зарядка их
    конденсаторов перед стробированием. Светодиодным вспышкам часто требуется период
    охлаждения после строба, в течение которого следующий строб будет невозможен. Это
    элемент управления, доступный только для чтения.

.. _v4l2-cid-flash-duration:

``V4L2_CID_FLASH_DURATION (integer)``
    Длительность импульса строба вспышки, генерируемого источником строба, при
    использовании внешнего строба. Этот элемент управления должен реализовываться
    устройством, генерирующим сигнал аппаратного строба вспышки, — обычно это сенсор
    камеры, подключённый к контроллеру вспышки.

    Для этого режима работы :ref:`источник строба <v4l2-cid-flash-strobe-source>`
    контроллера вспышки должен быть сконфигурирован в ``V4L2_FLASH_STROBE_SOURCE_EXTERNAL``.
    Для получения более подробной информации, пожалуйста, ознакомьтесь также с
    документацией там.

    По возможности единицей измерения должны быть микросекунды (мкс).

.. _v4l2-cid-flash-strobe-oe:

``V4L2_CID_FLASH_STROBE_OE (boolean)``
    Включает вывод сигнала аппаратного строба от источника строба при использовании
    внешнего строба. Этот элемент управления должен реализовываться устройством,
    генерирующим сигнал аппаратного строба вспышки, — обычно это сенсор камеры,
    подключённый к контроллеру вспышки.

    При условии, что драйвер устройства, генерирующего сигнал, поддерживает это, длину
    строб-сигнала можно сконфигурировать, регулируя его
    :ref:`длительность вспышки <v4l2-cid-flash-duration>`.

    Для этого режима работы :ref:`источник строба <v4l2-cid-flash-strobe-source>`
    контроллера вспышки должен быть сконфигурирован в ``V4L2_FLASH_STROBE_SOURCE_EXTERNAL``.
    Для получения более подробной информации, пожалуйста, ознакомьтесь также с
    документацией там.
