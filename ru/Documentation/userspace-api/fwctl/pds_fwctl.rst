=================
Драйвер fwctl pds
=================

:Author: Shannon Nelson

Обзор
=====

Устройство PDS Core предоставляет сервис fwctl через
auxiliary_device с именем pds_core.fwctl.N.  Драйвер pds_fwctl
привязывается к этому устройству и регистрирует себя в подсистеме
fwctl.  Получаемый в результате интерфейс пространства пользователя
используется приложением, входящим в состав программного пакета
AMD Pensando для Distributed Service Card (DSC).

Драйвер pds_fwctl почти ничего не знает о внутреннем устройстве
прошивки (firmware).  Он умеет лишь отправлять команды прошивке через
очередь сообщений pds_core для обработки запросов fwctl.  Набор
доступных операций fwctl зависит от прошивки в DSC, а версия приложения
пространства пользователя должна соответствовать прошивке, чтобы они
могли взаимодействовать друг с другом.

При создании соединения драйвер pds_fwctl запрашивает у прошивки
список конечных точек объектов прошивки, а для каждой конечной точки
драйвер запрашивает список операций для этой конечной точки.

Описание каждой операции включает определённый прошивкой атрибут
команды, который соответствует уровням области действия (scope) FWCTL.
Драйвер преобразует эти значения прошивки в значения области действия
FWCTL, которые затем могут использоваться для фильтрации
ограниченных по области действия пользовательских запросов.

Пользовательский API pds_fwctl
===============================

Каждый запрос RPC включает целевую конечную точку и идентификатор
операции, а также длины и указатели входного и выходного буферов.
Драйвер проверяет существование запрашиваемых конечной точки и
операций, а затем сверяет область действия запроса с требуемой областью
действия операции.  После этого запрос компонуется вместе с данными
запроса и отправляется прошивке через очередь сообщений pds_core, а
результаты возвращаются вызывающей стороне.

Конечные точки RPC, операции и содержимое буферов определяются
конкретным пакетом прошивки в устройстве, который различается в
зависимости от доступных конфигураций продукта.  Подробности доступны
в документации SDK для конкретного продукта.
