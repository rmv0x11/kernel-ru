==============================================
Специфичные для Linux особенности формата ELF
==============================================

Определения
===========

«Первым» заголовком программы (program header) считается тот, у которого
наименьшее смещение в файле: e_phoff.

«Последним» заголовком программы считается тот, у которого наибольшее
смещение в файле: e_phoff + (e_phnum - 1) * sizeof(Elf_Phdr).

PT_INTERP
=========

Первый заголовок программы PT_INTERP используется для определения имени файла
интерпретатора ELF. Остальные заголовки PT_INTERP игнорируются (начиная с
Linux 2.4.11).

PT_GNU_STACK
============

Последний заголовок программы PT_GNU_STACK определяет исполняемость стека в
пространстве пользователя (начиная с Linux 2.6.6). Остальные заголовки
PT_GNU_STACK игнорируются.

PT_GNU_PROPERTY
===============

Используется последний заголовок программы PT_GNU_PROPERTY интерпретатора ELF
(начиная с Linux 5.8). Если у интерпретатора такого заголовка нет, то
используется последний заголовок программы PT_GNU_PROPERTY исполняемого файла.
Остальные заголовки PT_GNU_PROPERTY игнорируются.
