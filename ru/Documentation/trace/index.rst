==========================================================
Руководство по технологиям трассировки в Linux
==========================================================

Трассировка в ядре Linux — это мощный механизм, который позволяет
разработчикам и системным администраторам анализировать и отлаживать
поведение системы. Это руководство содержит документацию по различным
фреймворкам и инструментам трассировки, доступным в ядре Linux.
